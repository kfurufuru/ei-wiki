# 自工場固有情報

> 当工場（沿岸埋立地・コンビナート区域）に特有の電気計装情報をまとめる。
> 汎用知識は他カテゴリを参照し、ここには**当工場固有の知見**のみを記録する。

## 当地区の特徴

| 項目 | 内容 |
|------|------|
| 立地 | 沿岸埋立地（四方を海水に囲まれた区域） |
| 海塩暴露 | 多方向から塩害を受ける |
| 法的区域 | 石油コンビナート等特別防災区域（政令指定） |
| 許認可窓口 | **2025年4月〜 高圧ガス製造許可は所轄市消防局**（旧：所轄都道府県） |
| 主要製品 | フォトレジスト用ポリマー、特殊重合性モノマー |
| 隣接工業地帯 | 臨海工業地帯（複数の素材・エネルギー事業者と混在） |
