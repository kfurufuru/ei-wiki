# タグ一覧

ei-wiki のタグは **統制語彙30語** に限定します。表記揺れや1回きりのタグを排除し、タグによる横断検索・分類を機能させるためです。新しいタグの追加は禁止です。記事作成時は以下から選び、**1ページ2〜4個**（分野1個必須＋主題0〜2個＋用途0〜1個）を付与してください。

このページは MkDocs tags プラグインの一覧ページとして機能し、下部に各タグでフィルタした記事一覧を自動生成します。

## 分野タグ（各ページ必ず1個）

所属ディレクトリから機械的に決まります。1ページに分野タグは1個だけです。

| タグ | 意味 |
|------|------|
| 高圧 | 高圧受変電・受電設備・保護継電器などの高圧設備 |
| 低圧 | 低圧配電・モーター・制御回路などの低圧設備 |
| 計装 | センサ・伝送器・DCS/PLC・制御弁などの計装・プロセス制御 |
| 設計 | 電気設計・仕様書・負荷計算・図面・更新設計 |
| 保全 | 点検・校正・診断・予兆保全などの保全業務 |
| トラブル | 現場トラブルの原因究明・対処（逆引き診断） |
| 工事管理 | 工事フロー・検収・試運転・調達などの工事管理 |
| 安全 | 作業許可・LOTO・リスクアセスメント・安全教育 |
| 法規・保安 | 電気事業法・保安規程・事故報告・主任技術者業務 |
| 省エネ | 省エネ・エネルギー管理・デマンド・省エネ法 |
| 生成AI | 生成AIの業務活用（ChatGPT・Copilot 等） |

## 用途タグ（記事の使われ方・0〜1個）

| タグ | 意味 |
|------|------|
| 手順書 | 作業手順・実務マニュアル・チェック手順 |
| チェックリスト | チェックリスト形式の点検・確認シート |
| 計算 | 計算式・計算ツール（負荷計算・短絡電流・電圧降下等） |

| 入門 | 入門・基礎・学習者向けガイド |

## 運用ルール

- 統制語彙30語のみ使用。新タグ追加・表記揺れ（例: 「PLC」「DCS」を「DCS・PLC」以外で書く）は禁止。
- 1ページ2〜4個。分野1個は必須、主題0〜2個、用途0〜1個。
- 順序は 分野 → 主題 → 用途。
- 1ページにしか付かない粒度の細かい語は付けず、本文検索に委ねる。
