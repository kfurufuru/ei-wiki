# 業務マニュアル（ガイドライン一覧）

電気計装チームの現場作業・試験・安全手続きに関する実務マニュアルです。  
若手エンジニアが**現場でスマホを見ながら使う**ことを想定して作成しています。

---

## このセクションの使い方

1. 作業前日に該当マニュアルを確認し、**準備物・資格要件**をチェック
2. 現場では手順番号を追いながら作業
3. 疑問点は先輩や電計チームリーダーに確認してから進める

!!! danger "絶対ルール"
    不明点があれば**作業を止めて確認**。判断に迷った状態での作業続行は禁止。

## 共通の注意事項

!!! warning "資格・教育の確認"
    - 高圧活線近接作業：特別教育修了者のみ
    - 低圧電気取扱：特別教育修了者のみ
    - 高所作業（2m以上）：特別教育修了者のみ
    - 資格要件は各マニュアルの「事前確認」欄で都度確認すること

    マニュアルの内容に誤り・変更がある場合は電計チームリーダーに報告してください。
