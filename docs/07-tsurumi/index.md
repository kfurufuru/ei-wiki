# 鶴見工場固有情報

> 三菱ケミカル 関東事業所 鶴見工場（横浜市鶴見区大黒町10-1）に特有の電気計装情報をまとめる。
> 汎用知識は他カテゴリを参照し、ここには**鶴見固有の知見**のみを記録する。

## 鶴見大黒地区の特徴

| 項目 | 内容 |
|------|------|
| 所在 | 横浜市鶴見区大黒町（埋立地・ほぼ島） |
| 海塩暴露 | 四方が海水に囲まれ、多方向から塩害を受ける |
| 法的区域 | 石油コンビナート等特別防災区域（政令指定） |
| 許認可窓口 | **2025年4月〜 高圧ガス製造許可は横浜市消防局**（旧：神奈川県） |
| 主要製品 | リソマックス（半導体フォトレジスト用ポリマー）、アクリルアミド |
| 隣接工業地帯 | 京浜臨海工業地帯（JFE、ENEOS等と混在） |
