# 耐圧試験・絶縁診断

## 30秒まとめ

絶縁抵抗測定は「数値の絶対値」より「経年変化のトレンド」が重要。PI 値 2.0 以上が良好の目安。直流耐圧試験後は必ず残留電荷の放電を確認してから接触する。

---

## 絶縁抵抗測定 判定基準一覧

設備種別ごとの絶縁抵抗の目安値（電技解釈・IEC・現場実務から）。メガー印加電圧・法定絶縁抵抗値・実務管理値の正典は [絶縁管理](../05-hozen/insulation-management.md) とし、本表はそれと同一基準で整理している。

| 設備種別 | 測定電圧（メガー） | 判定基準 |
|---------|---------------|---------|
| 低圧回路（対地電圧 150V 以下） | 500 V | 法定 0.1 MΩ 以上（電技省令58条）、実務は 1 MΩ 以上を良好 |
| 低圧回路（300V 以下・対地150V 超） | 500 V | 法定 0.2 MΩ 以上、実務は 1 MΩ 以上を良好 |
| 低圧回路（300V 超） | 500 V | 法定 0.4 MΩ 以上、実務は 1 MΩ 以上を良好 |
| 6.6kV 高圧機器 | 5000 V | 法定値の定めなし。トレンド管理（前回比の急低下に注意） |
| 高圧ケーブル（6.6kV） | 5000 V | トレンド管理。新品は数 GΩ、劣化で低下 |
| 高圧モーター | 5000 V | PI 値 ≥ 2.0（下表参照）・トレンド管理 |
| 変圧器巻線 | 5000 V | トレンド管理（新品時の 1/10 以下は要注意） |

!!! note "メガー印加電圧の統一基準"
    低圧回路＝500V メガー、高圧機器・ケーブル（6.6kV 級）＝5000V メガーを原則とする（AC100V 以下の弱電機器は 250V、用途により 1000V・2500V もあり）。詳細は [絶縁管理](../05-hozen/insulation-management.md) を参照。

!!! warning "絶縁抵抗は温度依存性が大きい"
    巻線温度が 10℃上昇すると絶縁抵抗は約 1/2 に低下する。
    測定値は温度記録と合わせて管理し、同条件（同温度）での値を比較すること。

---

## PI 値（成極指数）の計算と評価

PI 値は時間とともに絶縁体内の電荷分布が安定していくことを利用した劣化指標。

```
PI = 10分後の絶縁抵抗（MΩ） / 1分後の絶縁抵抗（MΩ）
```

| PI 値 | 判定 | 対応 |
|-------|------|------|
| 4.0 以上 | 優良 | そのまま継続使用 |
| 2.0〜4.0 | 良好 | そのまま継続使用 |
| 1.5〜2.0 | 要注意 | 清掃・乾燥を実施して再測定 |
| 1.0〜1.5 | 不良 | 使用停止・巻線清掃・乾燥処置 |
| 1.0 未満 | 危険 | 運転禁止・修理または交換 |

---

## 吸収比（1分値/30秒値）

PI 値の簡易版として使われる指標。

```
吸収比 = 1分後の絶縁抵抗（MΩ） / 30秒後の絶縁抵抗（MΩ）
```

| 吸収比 | 判定 |
|-------|------|
| 1.3 以上 | 良好 |
| 1.0〜1.3 | 要注意 |
| 1.0 未満 | 不良 |

---

## 直流耐圧試験の手順と注意事項

### 試験電圧の目安

竣工時（新設・更新時）の 6.6kV ケーブルの試験電圧は電技解釈 第15条に基づく法定値。最大使用電圧＝6600 × 1.15 / 1.1 ＝ 6900 V を基準に、交流試験電圧＝6900 × 1.5 ＝ 10350 V（約10.35 kV）、その直流代替（ケーブルのみ許容）＝交流試験電圧 × 2 ＝ 20700 V（約20.7 kV）となる。いずれも印加時間は 10 分。詳細計算は [高圧ケーブル](hv-cable.md) と同一値で統一している。

| 設備 | 区分 | 試験電圧 | 印加時間 |
|------|------|---------|---------|
| 6.6kV 高圧ケーブル | 竣工時 交流 | AC 約10.35 kV | 10 分 |
| 6.6kV 高圧ケーブル | 竣工時 直流 | DC 約20.7 kV（交流の2倍・ケーブルのみ） | 10 分 |
| 6.6kV 高圧ケーブル | 維持（経年診断） | DC 5〜10 kV 程度の段階印加（実務例・要確認） | 社内基準 |
| 6.6kV モーター | 竣工時 交流 | AC 約10.35 kV | 10 分 |
| 6.6kV モーター | 維持（診断） | DC 5〜10 kV 程度（要確認） | 社内基準 |

!!! warning "維持診断の試験電圧・直流耐圧の非推奨"
    竣工時（新設・更新）の交流 10.35 kV／直流 20.7 kV・各10分は電技解釈 第15条の法定値。一方、経年ケーブルの**維持診断**の試験電圧は法定でなく、単一の正典値がないため JEAC 8021 等の一次資料・社内基準・メーカー仕様で必ず確認すること。全電圧印加は健全部を痛める恐れがあるため実務では低減電圧の段階印加が一般的。判定は絶対値でなく前回比（目安：前回の 80% 以下で劣化疑い）で行う。なお XLPE（CV）ケーブルへの直流耐圧は電界分布の相違・水トリー助長・残留電荷によるアフターブレークダウンの理由で近年非推奨となり、VLF（0.1Hz）耐圧・交流耐圧や tanδ・部分放電・残留電荷法などの非破壊診断へ移行している（ただし第15条の直流代替規定は法的には現存）。

### 試験手順

```
1. 試験対象を系統から完全に切り離す（断路器・VCB を開路）
2. 対象機器の両端に放電棒で接触し残留電荷がないことを確認
3. 高電圧試験器を接続（非試験側は接地）
4. ゆっくり電圧を上昇させる（急上昇はサージを与える）
5. 試験電圧到達後、所定時間（1〜10分）維持
6. 漏れ電流を測定・記録
7. 電圧をゆっくりゼロまで降下させる
8. 接地棒を使って放電する（★必ず実施）
9. 放電後も数分待ってから触れる
```

!!! danger "残留電荷の放電確認は最重要"
    直流耐圧試験後のケーブルや機器には試験電圧に近い残留電荷が蓄積されている。
    接地棒をケーブル端末に接触させると放電音がする場合がある。
    **放電確認なしに触れることは厳禁。**
    接地操作後も最低2〜3分は放電を継続し、接地したまま作業する。

## 測定前の残留電荷放電確認

あらゆる絶縁測定・耐圧試験の前にも、前回の試験や通常運転から残留している電荷を放電することが必要。

```
確認手順：
1. 測定対象を停電・切り離し
2. 接地棒（放電棒）を使って各端子を接地する
3. 接地した状態で最低30秒〜2分待つ
4. 残留電圧が0Vであることを検電器で確認
5. 測定器を接続する
```

!!! note "大容量コンデンサ・長距離ケーブルの放電時間"
    進相コンデンサや長距離高圧ケーブルは容量が大きいため、放電に時間がかかる。
    接地棒での放電後に放電抵抗（放電専用の高抵抗器）を使って安全に放電する。
