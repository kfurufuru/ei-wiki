# 保護リレー試験手順

過電流継電器（OCR）・地絡継電器（GR/DGR）などの保護リレーの動作試験手順です。  
年次点検・リレー更新後・設定変更後に実施します。

---

## 事前確認

!!! danger "停電作業が前提"
    保護リレー試験は原則として**停電状態**で実施する。  
    活線状態でのCT二次側開放は絶対禁止（高電圧発生・CT焼損）。

!!! warning "作業開始前に必ず確認"
    - [ ] PTW（作業許可証）取得済み
    - [ ] 停電作業計画書・スケジュール製造課と合意済み
    - [ ] 試験用継電器試験器（リレーテスタ）の準備・校正確認
    - [ ] 動作試験記録シート準備
    - [ ] 対象リレーの**保護協調図・整定値一覧**入手済み

### 必要な資格
- 電気工事士（第1種）または電気主任技術者の指導のもと作業
- 高圧電気取扱特別教育修了者

### 必要な機器
| 機器 | 用途 |
|------|------|
| 継電器試験器（リレーテスタ） | 試験電流・電圧の発生 |
| 秒時計・タイマー機能付き | 動作時間計測 |
| デジタルマルチメータ | CT二次電流・電圧確認 |
| 保護協調図・整定値一覧 | 試験条件確認 |

!!! warning "試験中に異常を検知したら（初動）"
    1. CT二次側の開放・異音・発煙・異常発熱などを認めたら、**直ちに試験電流を停止し作業を中断**する
    2. **作業指揮者へただちに報告**する
    3. 停電範囲・復電可否を**電気主任技術者と協議**する
    4. 原因確認と安全確保ができるまで**試験を再開しない**
    （報告先・連絡順序・時間の目安は社内規定・作業計画書に従う）

---

## 主な保護リレーの種類と試験目的

| リレー種別 | 略称 | 目的 |
|-----------|------|------|
| 過電流継電器 | OCR (51) | 短絡・過負荷保護 |
| 地絡継電器 | GR (64) | 一般地絡保護 |
| 地絡方向継電器 | DGR (67G) | 方向性地絡保護（非接地系統） |
| 過電圧継電器 | OVR (59) | 過電圧保護 |
| 不足電圧継電器 | UVR (27) | 停電・低電圧検出 |
| 距離継電器 | Dist (21) | 送電線保護 |

## 手順

### STEP 1：作業前安全確認

1. 対象回路の**停電・検電・接地**を確認する
    - 検電器で充電なしを確認
    - 接地棒を取り付けて誤通電を防ぐ
2. CT（変流器）二次側の短絡端子を確認する
    - 試験中にCT二次を開放しないこと
    - 試験前に短絡ジャンパを取り付けてから回路を切り離す
3. 保護協調図から整定値を確認し、試験条件を記録シートに転記する

### STEP 2：リレーテスタの接続

#### OCR（過電流継電器）の場合

1. CT二次側にリレーテスタの電流出力端子を接続する
2. リレーの動作出力端子（接点）にテスタを接続して動作検出できるようにする
3. リレーテスタの電流値を**整定電流（Tap電流）**に合わせて設定する

!!! info "Tap電流の確認"
    OCR整定票の「Tap」欄を確認する。例：Tap=5A、CT比=100/5  
    整定電流（一次換算）= Tap × CT比 = 5A × 20 = 100A

#### GR・DGR（地絡継電器）の場合

1. ZCT（零相変流器）二次側にリレーテスタの電流出力を接続
2. DGRの場合は電圧方向要素への接続も必要（試験器の電圧出力も使用）

### STEP 3：動作試験

#### 最小動作電流（感度）試験

1. 整定電流値の**105%相当の電流**を徐々に増加させながら入力する
2. リレーが動作した電流値を記録する
3. **判定基準**：整定値の ±10% 以内（機器仕様書で確認）

| 試験電流 | 正常な動作 | NG（不合格）と理由 |
|---------|---------|------|
| 整定値 × 90% | 不動作が正常 | 動作したらNG（感度が過敏） |
| 整定値 × 105% | 動作が正常 | 不動作ならNG（感度不足） |

#### 動作時間試験

1. 整定電流値の**200%・300%相当の電流**を入力する（反限時特性の確認）
2. 電流入力から接点出力までの時間をタイマーで計測する
3. 時限曲線（T-I特性図）と照合して許容範囲内であることを確認する

!!! info "反限時と定限時"
    - **反限時（IDMT）**：電流が大きいほど動作時間が短い。IEC 60255規格の特性曲線で確認
    - **定限時**：電流値によらず整定時間で動作。直接整定値と比較

**動作時間の判定基準**：整定値 ±10% 以内（または機器仕様書に従う）

#### 瞬時要素試験（High-Set/Inst.）

1. 瞬時整定値の**110%相当の電流**を入力し、100ms以内に動作することを確認
2. 瞬時整定値の**90%相当の電流**を入力し、不動作であることを確認

### STEP 4：復帰試験

1. リレー動作後、電流を徐々に下げる
2. **復帰電流値**（リセット電流）を記録する
3. 復帰比（復帰電流/動作電流）を確認する
    - 通常 90%以上（低すぎると微小変動でハンチング）

### STEP 5：接点確認

1. 動作接点（a接点・b接点）の動作を確認する
    - 遮断器トリップ回路の動作確認
    - 警報出力の動作確認
2. 接点の**接触抵抗**をマルチメータで確認する（0.1Ω以下が目安）

### STEP 6：現地整定値の確認

1. リレー本体の整定スイッチ・ダイヤルを確認し、整定値一覧と一致していることを確認
2. デジタルリレーの場合は設定パラメータをリードアウトして記録
3. 前回点検からの変更の有無を確認

### STEP 7：復旧・記録

1. CT二次側短絡ジャンパを取り外す（接続を元通りに戻す）
2. 接地棒を取り外す
3. 動作試験記録シートに全測定値・判定結果を記録
4. チームリーダー・電気主任技術者のサインを得る

## チェックリスト（作業完了時）

- [ ] 停電・検電・接地の確認完了
- [ ] CT二次側開放防止措置（短絡ジャンパ）の取付・取外し管理完了
- [ ] 感度試験（最小動作電流）：判定 OK
- [ ] 動作時間試験（反限時/定限時）：判定 OK
- [ ] 瞬時要素試験：判定 OK（瞬時設定がある場合）
- [ ] 復帰試験完了
- [ ] 接点動作確認（トリップ・警報）完了
- [ ] 整定値と本体設定値の一致確認完了
- [ ] CT二次側の復旧確認（開放状態でないこと）
- [ ] 試験記録シートへの記録・サイン完了

## 判定基準まとめ

| 試験項目 | 合格基準 |
|---------|---------|
| 最小動作電流 | 整定値 ±10% |
| 動作時間 | 整定値 ±10% または仕様書値 |
| 瞬時動作時間 | 100ms 以内 |
| 復帰比 | 90% 以上 |
| 接点接触抵抗 | 0.1Ω 以下 |

---

## よくある不具合と対処

| 不具合 | 原因 | 対処 |
|--------|------|------|
| 感度ずれ（整定値より早期動作） | 整定スイッチ接触不良 | 整定スイッチ清掃・締め直し |
| 動作時間が長い | 機構摩耗（電磁式） | 調整または交換検討 |
| 接点不動作 | 接点酸化・汚損 | 接点清掃または交換 |
| DGRが電流のみで動作 | 方向要素の電圧入力断 | PT二次配線確認 |
