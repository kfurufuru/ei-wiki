# 耐圧試験・絶縁診断

## 30秒まとめ

絶縁抵抗測定は「数値の絶対値」より「経年変化のトレンド」が重要。PI 値 2.0 以上が良好の目安。直流耐圧試験後は必ず残留電荷の放電を確認してから接触する。

---

## 絶縁抵抗測定 判定基準一覧

設備種別ごとの絶縁抵抗の目安値（電技解釈・IEC・現場実務から）。

> 絶縁抵抗計の定格電圧は測定対象で選ぶ：低圧回路は 500V（250/1000V も）、高圧機器・ケーブルは 1000〜2500V が一般的、特別高圧・長尺高圧ケーブルで 5000V。電圧が不足すると絶縁不良を「良好」と誤判定しうる。

| 設備種別 | 測定電圧（メガー） | 判定基準 |
|---------|---------------|---------|
| 低圧回路（300V 以下） | 500 V | 0.2 MΩ 以上（最低基準・電技省令第58条）、実務的には 1 MΩ 以上 |
| 低圧回路（300V 超） | 500〜1000 V | 0.4 MΩ 以上（電技省令第58条）、実務的には 10 MΩ 以上 |
| 6.6kV 高圧機器 | 1000〜2500 V | 1000 MΩ（1 GΩ）以上が良好の目安 |
| 高圧ケーブル（6.6kV） | 1000〜2500 V | 2000 MΩ 以上（劣化が進むと低下） |
| 高圧モーター | 1000 V | PI 値 ≥ 2.0（表参照） |
| 変圧器巻線 | 1000〜2500 V | 2000 MΩ 以上（新品時の 1/10 以下は要注意） |

!!! warning "絶縁抵抗は温度依存性が大きい"
    巻線温度が 10℃上昇すると絶縁抵抗は約 1/2 に低下する。
    測定値は温度記録と合わせて管理し、同条件（同温度）での値を比較すること。

!!! note "絶縁抵抗：法定下限と実務目安は別物"
    低圧電路の絶縁抵抗の **法定下限** は電技省令第 58 条で対地電圧により定められる（開閉器または過電流遮断器で区切れる電路ごとに測定）。

    | 対地電圧 | 法定下限 |
    |---|---|
    | 150V 以下 | 0.1MΩ 以上 |
    | 150V 超〜300V 以下 | 0.2MΩ 以上 |
    | 300V 超 | 0.4MΩ 以上 |

    実務でよく使う「**1MΩ 以上で合格**」は法定値より厳しい **自主管理目安** であって法律上の下限ではない。0.1〜0.4MΩ を割れば法令違反、1MΩ を割ったら（法定下限まで余裕はあるが）早期に原因調査、という二段で捉える。

## PI 値（成極指数）の計算と評価

PI 値は時間とともに絶縁体内の電荷分布が安定していくことを利用した劣化指標。

```
PI = 10分後の絶縁抵抗（MΩ） / 1分後の絶縁抵抗（MΩ）
```

| PI 値 | 判定 | 対応 |
|-------|------|------|
| 4.0 以上 | 優良 | そのまま継続使用 |
| 2.0〜4.0 | 良好 | そのまま継続使用 |
| 1.5〜2.0 | 要注意 | 清掃・乾燥を実施して再測定 |
| 1.0〜1.5 | 不良 | 使用停止・巻線清掃・乾燥処置 |
| 1.0 未満 | 危険 | 運転禁止・修理または交換 |

---

## 吸収比（1分値/30秒値）

PI 値の簡易版として使われる指標。

```
吸収比 = 1分後の絶縁抵抗（MΩ） / 30秒後の絶縁抵抗（MΩ）
```

| 吸収比 | 判定 |
|-------|------|
| 1.3 以上 | 良好 |
| 1.0〜1.3 | 要注意 |
| 1.0 未満 | 不良 |

## 直流耐圧試験の手順と注意事項

### 試験電圧の目安

| 設備 | 試験電圧（参考） |
|------|--------------|
| 6.6kV 高圧ケーブル（竣工時） | 22〜25 kV 直流 |
| 6.6kV 高圧ケーブル（維持） | 10〜15 kV 直流（劣化状態に応じて） |
| 6.6kV モーター（竣工時） | 10 kV 直流 |
| 6.6kV モーター（維持） | 5〜8 kV 直流 |

（参考値。機器仕様・社内基準・メーカー推奨に従うこと）

### 試験手順

```
1. 試験対象を系統から完全に切り離す（断路器・VCB を開路）
2. 対象機器の両端に放電棒で接触し残留電荷がないことを確認
3. 高電圧試験器を接続（非試験側は接地）
4. ゆっくり電圧を上昇させる（急上昇はサージを与える）
5. 試験電圧到達後、所定時間（1〜10分）維持
6. 漏れ電流を測定・記録
7. 電圧をゆっくりゼロまで降下させる
8. 接地棒を使って放電する（★必ず実施）
9. 放電後も数分待ってから触れる
```

!!! danger "残留電荷の放電確認は最重要"
    直流耐圧試験後のケーブルや機器には試験電圧に近い残留電荷が蓄積されている。
    接地棒をケーブル端末に接触させると放電音がする場合がある。
    **放電確認なしに触れることは厳禁。**
    接地操作後も最低2〜3分は放電を継続し、接地したまま作業する。

### 試験中の異常・接触事故時の連絡

漏れ電流の急増・絶縁破壊・感電疑いなどの異常が発生した場合は、次の順序で対応する。

!!! danger "異常・事故発生時の対応フロー"
    **① 直ちに昇圧を停止し、電源を遮断する。**
    **② 対象を接地して残留電荷を放電する（安全確保が最優先）。**
    ③ 試験責任者（電気主任技術者）へ速やかに報告し、試験の中断・再開を判断する。
    ④ 感電・負傷がある場合は救護を最優先し、必要に応じて 119 番通報する。
    ⑤ 事故・重大異常は社内保安部門へ連絡し、社内規定に従って所定の事故報告手続きを行う。

    連絡先・報告期限・事故報告の要否は社内保安規程および関係先の定めに従うこと。

## 測定前の残留電荷放電確認

あらゆる絶縁測定・耐圧試験の前にも、前回の試験や通常運転から残留している電荷を放電することが必要。

```
確認手順：
1. 測定対象を停電・切り離し
2. 接地棒（放電棒）を使って各端子を接地する
3. 接地した状態で最低30秒〜2分待つ
4. 残留電圧が0Vであることを検電器で確認
5. 測定器を接続する
```

!!! note "大容量コンデンサ・長距離ケーブルの放電時間"
    進相コンデンサや長距離高圧ケーブルは容量が大きいため、放電に時間がかかる。
    接地棒での放電後に放電抵抗（放電専用の高抵抗器）を使って安全に放電する。
