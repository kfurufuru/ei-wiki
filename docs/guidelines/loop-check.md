# 計装ループチェック手順

新設・改造工事後に実施する計装ループの動作確認手順です。  
DCS/調節計への信号入力から最終制御要素（制御弁・ポンプ等）までの一連動作を確認します。

---

## 事前確認

!!! warning "作業開始前に必ず確認"
    - [ ] PTW（作業許可証）取得済み
    - [ ] 対象ループの P&ID・計装図（ループ図）準備済み
    - [ ] 製造課との調整完了（プロセスへの影響確認）
    - [ ] 無線機または内線電話の準備（DCS室↔現場の連絡手段）
    - [ ] 校正済み電流発生器（ハンドヘルドキャリブレータ）準備

### 必要な資格
- 低圧電気取扱特別教育修了（電源ON/OFF作業時）
- 計装工事経験者の立会い（初回作業者）

### 必要な工具・機器
| 機器 | 用途 |
|------|------|
| ハンドヘルドキャリブレータ（HART対応推奨） | 信号発生・センサシミュレーション |
| デジタルマルチメータ | 電圧・抵抗測定 |
| ループ図・配線図 | 配線確認 |
| トランシーバ or 内線電話 | DCS室との連絡 |
| 記録用紙（ループチェックシート） | 測定値記録 |

---

## ループチェックの流れ

```
センサ模擬信号入力 → フィールド信号確認 → DCS表示確認 → 制御出力確認 → 最終制御要素動作確認
```

---

## 手順

### STEP 1：ループ図の確認

1. 対象ループの**ループ図**を入手し、以下を確認する
    - センサ種別（熱電対・測温抵抗体・4-20mA発信器・差圧発信器 など）
    - 測定範囲（スパン）と単位
    - 制御出力先（制御弁・インバータ・ポンプ など）
    - 電源供給元（ターミナルボックス番号・端子番号）
2. DCS/調節計のタグ番号を確認する（例：FIC-101、TIC-202）

### STEP 2：配線確認（導通チェック）

1. センサ側の端子台でマルチメータを使い、**芯線の導通**を確認する
    - 断線の有無：抵抗値が∞の場合は断線
    - 極性の確認：＋と－が逆転していないか
2. シールド線の接地状態を確認する
    - 片端接地（受信側 or 発信側のどちらか一方）になっているか
    - 両端接地になっている場合は電計TLに報告

!!! warning "絶縁確認"
    配線導通確認後、DCSへの接続前に**芯線間・芯線-アース間の絶縁抵抗**を測定する。  
    基準値：DC500V メガーで **10MΩ以上**（低い場合は工事担当者に報告）

### STEP 3：信号発生（センサシミュレーション）

#### 4-20mA 発信器の場合

1. センサを端子台から切り離す（または発信器のループ断点で接続）
2. キャリブレータを接続し、以下の3点で電流を発生させる

| 発生電流 | 相当する入力値 | DCS表示確認目標 |
|---------|--------------|---------------|
| 4.00 mA | 0% （スパン下限） | 測定範囲の下限値 |
| 12.00 mA | 50%（スパン中央） | 測定範囲の中央値 |
| 20.00 mA | 100%（スパン上限） | 測定範囲の上限値 |

3. DCS室の担当者に各値での表示を口頭または記録シートで報告・確認してもらう
4. **誤差基準**：表示誤差 ±0.5% of Full Scale（機器仕様書で確認）

#### 熱電対（TC）・測温抵抗体（RTD）の場合

1. センサを切り離してキャリブレータを接続
2. キャリブレータでTC/RTD種別を選択し、以下の温度を模擬入力する
    - 下限温度・中央温度・上限温度の3点
3. DCS表示値が設定温度と一致することを確認

#### 差圧発信器（流量・圧力）の場合

1. 発信器のゼロ点（均圧弁開・高低圧弁閉）でDCS表示が0%になることを確認
2. キャリブレータで4-20mA を模擬入力して上記と同手順で確認

### STEP 4：アラーム・インターロック確認

1. DCS室担当者に確認してもらいながら、アラーム設定値付近の信号を入力する
2. 以下の動作を確認する
    - [ ] HH（高高）アラーム発報
    - [ ] HI（高）アラーム発報
    - [ ] LO（低）アラーム発報
    - [ ] LL（低低）アラーム発報
    - [ ] インターロック（トリップ）動作（設定がある場合）

!!! danger "インターロック試験の注意"
    インターロック試験はプロセスへの影響が大きい。  
    必ず製造課・運転員の立会い・承認のもとで実施すること。  
    バイパス設定が必要な場合は製造課長の承認を得ること。

### STEP 5：制御出力確認

1. DCS室から制御出力を手動で操作してもらう
2. 現場でアクチュエータ（制御弁・インバータ等）の動作を確認する

| 出力値 | 確認内容 |
|-------|---------|
| 0% | 制御弁：全閉 / インバータ：停止 |
| 50% | 制御弁：半開 / インバータ：50%回転数 |
| 100% | 制御弁：全開 / インバータ：最大回転数 |

3. 動作方向（フェールセーフ方向）を確認する
    - FA（フェールオープン）：信号喪失時に全開
    - FC（フェールクローズ）：信号喪失時に全閉

### STEP 6：記録・サインオフ

1. ループチェックシートに全測定値・確認結果を記録する
2. 異常がなければ「OK」、不具合があれば「NG」と判定し原因・対策を記載
3. 現場担当者・DCS室担当者・電計チームリーダーのサインをもらう
4. 記録は工事書類として保管する

---

## チェックリスト（作業完了時）

- [ ] 全ての信号点で DCS 表示値が正常範囲内
- [ ] アラーム動作確認済み（HH/HI/LO/LL）
- [ ] インターロック動作確認済み（該当ループのみ）
- [ ] 制御弁・アクチュエータの動作方向確認済み
- [ ] シールド接地状態確認済み
- [ ] 絶縁抵抗値記録済み
- [ ] ループチェックシートへの記録・サイン完了
- [ ] センサ・端子台の復旧（仮接続の解除）

---

## よくある不具合と対処

| 不具合 | 原因 | 対処 |
|--------|------|------|
| DCS表示が動かない | 断線・端子緩み | 配線確認・端子増し締め |
| 表示値がずれる（オフセット） | ゼロ点ずれ・配線極性誤り | 極性確認・発信器ゼロ調整 |
| アラームが出ない | 設定値確認・入力信号不足 | DCS設定値・配線確認 |
| 制御弁が動かない | 空気源圧力不足・ポジショナ故障 | 空気圧確認・ポジショナ点検 |
| 信号がノイズで振れる | シールド接地不良・誘導 | シールド接地見直し |
