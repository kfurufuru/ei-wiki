# 電気計装工事で押さえるJIS規格

## 30秒まとめ

計装工事で出てくるJISは **図記号・信号・温度センサ・圧力流量・保護等級・ケーブル・防爆** のグループに整理できます。番号を暗記するより、**「その規格が何を決めていて、工事のどこに効くか」** を押さえるのが実務的です。計装のJISは IEC を国内に取り込んだもの（C で始まる5桁番号など）が多く、現場では JIS に加えて ISA・NAMUR が事実上の標準として併用されます。

!!! info "このページの位置づけ — 一覧・法規との役割分担"
    - 本ページ: 計装現場での **使いどころ**（何に効くJISか）
    - [規格一覧](../reference/standards-list.md): 規格番号・タイトル・**入手先**の網羅表
    - [規格・法規](../04-sekkei/standards.md): 電気事業法・高圧ガス保安法・消防法など **化学プラント法規**

## A. 図記号・図面

図面を読み書きするための記号の規格です。電気側と計装側で別系統の記号体系を使う点に注意します。

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS C 0617 | 電気用図記号（IEC 60617 対応） | 展開図・結線図・単線結線図の図記号。旧 JIS C 0301 から置き換わっており、古い図面は旧記号のことがある |
| JIS Z 8204:1983 | 計装用記号 | P&ID・計装フロー図のループ番号・機能記号（TT/TIC など）。ISA-5.1 を基にした和製版で、2024年時点で有効。ただし会社ごとに自社標準を持つことが多く、現場の図面凡例を必ず確認 |

!!! tip "電気記号と計装記号は別物"
    同じ1枚の図面でも、電気部分は JIS C 0617、計装ループは JIS Z 8204（または社内標準・ISA-5.1）で描かれます。記号の出典を取り違えると読み違いにつながります。

---

## B. 信号（計装の心臓部）

伝送器から DCS・PLC までを結ぶアナログ信号・デジタル通信の規格です。

| 規格 | 何を決めるか | 工事での要注意点 |
|------|------------|----------------|
| IEC 60381-1 | 計装用統一信号 DC 4-20 mA（DC 電流信号） | 伝送器〜受信計器間の標準アナログ信号。**対応するJIS番号は無く、実務はIECに準拠** |
| NAMUR NE 43 | 4-20 mA の障害検出レベル（3.6 mA 以下／21 mA 以上） | 断線・センサ異常時に信号をどちらへ振るか（フェイル方向）の設計根拠 |
| IEC 61158 / IEC 61784 | フィールドバス（FOUNDATION Fieldbus・PROFIBUS 等）、HART 重畳通信 | デジタル通信計装の配線・トポロジ・終端処理 |

!!! warning "「統一信号のJIS番号」は存在しない"
    4-20 mA 統一信号にきれいなJIS番号を当てるのは誤りです。実体は **IEC 60381-1**（DC 電流信号）であり、フェイル検出の考え方は **NAMUR NE 43** が事実上の標準です。仕様書に根拠規格を書くときはここを混同しないようにします。

> 信号方式そのものの基礎は [計装基礎](basics.md) を参照してください。

## C. 温度計測

計装JISの中核グループです。センサの特性と許容差クラス、補償導線の規格が工事の精度・誤接続防止に直結します。

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS C 1602 | 熱電対（K・J・T・E・R・S・B・N 型） | 型式ごとの起電力特性・許容差クラス（クラス1/2/3）。型式の取り違えはそのまま指示誤差になる |
| JIS C 1604 | 測温抵抗体（Pt100 など） | クラスA／クラスB の許容差、3線式/4線式の結線。配線抵抗が誤差に効くため結線方式の選定が重要 |
| JIS C 1605 | シース熱電対 | 保護管径・応答性・絶縁の選定 |
| JIS C 1610 | 熱電対用補償導線 | 補償導線の種類記号（KX・JX 等）と識別色。**逆接続・銅線代用・途中材質切替は代表的な現場ミス** |

> センサの選び方・配線方式・補償導線の誤接続対策は [温度計測](temperature.md) で詳しく扱っています。

## D. 圧力・流量

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS B 7505-1 | ブルドン管圧力計 | 現場指示計の精度等級・目盛・接続。レンジは常用圧が目盛中央付近に来るように選定 |
| JIS Z 8762 | 円形管路の絞り機構による流量測定（オリフィス等、ISO 5167 対応） | 差圧式流量計の絞り板設計・上下流の必要直管長。直管長不足は計測誤差の定番原因 |

> 計測方式ごとの選定は [圧力計測](pressure.md)・[流量計測](flow.md) を参照してください。

## E. 機器の保護等級・設置環境

フィールド機器をどの環境に置けるかを決める保護等級の規格です。屋外・洗浄環境の多い化学プラントでは選定ミスが浸水・故障に直結します。

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS C 60529（旧 JIS C 0920） | 外郭による保護等級（IP コード） | フィールド機器・端子箱・盤の防塵防水等級（IP65・IP67 など）。設置環境（屋外・薬液飛散・洗浄）に対し等級が足りているかを確認 |
| JIS C 60079 系 | 防爆（耐圧 Ex d・安全増 Ex e・本質安全 Ex i・危険場所への設置） | 計装では **本質安全（Ex i）** が主役。バリア・配線・機器の組み合わせ、危険場所区分との整合を確認 |

!!! warning "IP コードの規格番号が新しくなりました（2026年1月）"
    IP コードは **2026年1月20日付で JIS C 0920:2003 → JIS C 60529 に移行**しました（IEC 60529 と整合させ番号を5桁化、高温・高圧水に対する **IPX9** を追加）。既存の機器銘板・仕様書・図面の多くは当面 **旧番号 JIS C 0920 や IP 表記のまま**なので、**新旧どちらの番号も同じIPコード体系を指す**ことを理解しておきます。

> 防爆構造・エリア区分の詳細は [防爆](explosion-proof.md) を参照してください。

## F. ケーブル・配線・接地

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS C 3401 | 制御用ケーブル（CVV：制御用ビニル絶縁ビニルシースケーブル） | 制御・信号配線の選定。許容電流より絶縁・芯数・機械的保護で選ぶことが多い |
| JIS C 0364 | 建築電気設備（低圧電気設備、IEC 60364 対応） | 低圧配線・保護・接地の基本設計 |

- 計装ケーブルはツイストペア・シールド（遮へい）付きを使い、シールドの **片端接地** などノイズ対策の処理が肝心です。配線・シールド処理の実務は [計装配線](wiring.md) を参照してください。
- 計装の「GND」と保安の「接地」は別概念です。区別と種類（FG/SG/AGND/DGND/PGND）は [グランドと接地](grounding-gnd.md) を参照してください。

## G. 制御盤・操作機器

| 規格番号 | 何を決めるか | 工事での要注意点 |
|---------|------------|----------------|
| JIS C 8201 | 低圧開閉装置及び制御装置（IEC 60947 対応） | 盤内の遮断器・接触器・スイッチ類の定格・性能 |
| JIS C 0448 | 表示・操作機器の識別（色・記号） | 押しボタン・表示灯の色（赤=停止/異常、緑=運転 など）。非常停止は赤・黄地が基本 |
| JIS C 1010-1 | 測定・制御・試験室用電気機器の安全 | 計測器・変換器など機器側の安全要求 |

> 盤の構造設計は [盤設計](../04-sekkei/panel-design.md)、低圧側の制御回路は [制御回路](../02-teiatsu/control-circuit.md) を参照してください。

## H. 安全計装・機能安全

| 規格 | 何を決めるか | 工事での要注意点 |
|------|------------|----------------|
| IEC 61511 | プロセス産業の安全計装システム（SIS） | 緊急遮断（ESD）・SIL の設計と検証。化学プラントでの安全計装の中心規格 |
| IEC 61508 | 電気・電子・プログラマブル安全関連系の機能安全 | SIL の定義となる基礎規格 |

- 安全計装は一般のループと **保守・試験・変更管理（MOC）の扱いが異なる**ため、対象ループが SIS かどうかを工事前に確認します。
- 計量器の校正（取引・証明用）は計量法・JCSS の体系で、保安の安全計装とは別管理です。

---

## 計装工事でのJIS活用のコツ

- **番号より「何を決める規格か」で覚える**: 仕様書・指摘事項で根拠を示すとき、温度なら C 1602/C 1604、IP なら C 60529、図記号なら C 0617 と引ければ十分です。
- **JIS と ISA・NAMUR を混同しない**: 計装記号・統一信号・アラーム管理は ISA や NAMUR が事実上の標準です。JIS で書ける部分とそうでない部分を切り分けます。
- **新旧番号の併存に注意**: IP コード（C 0920→C 60529）のように番号が変わった規格は、現場の銘板・図面と仕様書で番号が食い違って見えることがあります。
