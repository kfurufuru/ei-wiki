# 接地抵抗測定手順

設備接地の定期検査・法定点検における接地抵抗測定手順です。  
電気設備技術基準により、接地工事の種類ごとに抵抗値が定められています。

---

## 事前確認

!!! warning "作業開始前に必ず確認"
    - [ ] PTW（作業許可証）取得済み
    - [ ] 接地抵抗計の校正有効期限確認（校正シールで確認）
    - [ ] 補助接地極の打ち込み場所確認（測定対象の接地極から十分離れた位置）
    - [ ] 雨天直後の測定を避ける（土壌水分が多く低い値が出る）
    - [ ] 接地対象設備の電源OFF確認（漏電電流による誤差防止）

### 必要な機器

| 機器 | 仕様目安 |
|------|---------|
| 接地抵抗計（アーステスタ） | JIS C 1304準拠、0.01Ω〜1kΩ以上のレンジ |
| 補助接地棒（P棒・C棒） | 各1本（長さ30cm以上の金属棒） |
| 補助接地コード | 各10m以上（色分け推奨：黄・緑） |
| スパナ・ドライバ | 接地端子取り外し用 |
| 記録用紙 | 測定値記録用 |

---

## 接地工事の種別と基準値

| 接地工事の種類 | 対象設備 | 接地抵抗値 |
|--------------|---------|-----------|
| A種接地工事 | 高圧機器の外箱、避雷針 | **10Ω以下** |
| B種接地工事 | 変圧器二次側中性点 | 150/Ig Ω以下（Ig：1線地絡電流[A]） |
| C種接地工事 | 300V超の低圧機器外箱 | **10Ω以下**（漏電遮断器設置時は500Ω以下） |
| D種接地工事 | 300V以下の低圧機器外箱 | **100Ω以下**（漏電遮断器設置時は500Ω以下） |

> 出典: 接地工事の種類・抵抗値、および0.5秒以内に動作する漏電遮断器設置時の500Ω緩和は電技解釈第17条。

!!! info "B種接地抵抗の計算"
    B種接地抵抗 = 150 ÷ Ig  
    例：Ig = 5A → 150 ÷ 5 = 30Ω以下  
    Ig は電力会社への問い合わせまたは構内1線地絡電流から算出

---

## 測定方法：3極法（標準法）

最も一般的な測定方法。補助接地棒2本を使用します。

### 配置のルール

```
[測定対象接地極 E] ←── 10m ──→ [補助P極] ←── 10m ──→ [補助C極]
```

- E（被測定接地極）〜P（電位補助）〜C（電流補助）を**一直線上**に配置
- E〜C間：**20m以上**（推奨）
- E〜P間：E〜C間の**50%の位置**

*P極をE〜C間の50%付近（平坦部）に置くと、電位降下カーブが平らになり真の接地抵抗を読み取れる。E極やC極に寄せると値が急変し誤差が大きくなる。*

### 手順

1. 補助接地棒（P棒・C棒）を所定の位置に打ち込む
    - 地面に対して垂直に、ハンマーで深さ20cm以上打ち込む
    - 石・コンクリートの上は避ける
2. 接地抵抗計のリード線を接続する
    - E端子（通常：緑）→ 被測定接地端子
    - P端子（通常：黄）→ 補助P棒
    - C端子（通常：赤）→ 補助C棒

!!! warning "接地端子の切り離し"
    測定対象の接地線を**他の接地系統から切り離して**から測定する。  
    切り離さないと並列接地の合成抵抗を測定することになり、正しい値が得られない。

3. 接地抵抗計のスイッチを押して測定する
    - 指針またはデジタル値が安定するのを待ってから読み取る
    - 3回測定して平均値を記録する（ばらつきが大きい場合は補助極の位置を変えて再測定）

4. 測定終了後、接地端子を元通りに接続する（**必ず復旧を確認**）

## 測定方法：2極法（簡易法）

補助接地棒を打ち込む場所がない場合（コンクリート床上等）に使用。  
精度は3極法より劣る。

1. 接地抵抗計のE端子を被測定接地端子に接続
2. P・C端子を**同一の補助極（既設の別接地）**に接続
3. 別接地の抵抗値が十分低い（ほぼ0Ω）ことが前提
4. 測定値 = 被測定接地抵抗 + 補助極抵抗（補助極が低いほど精度が上がる）

!!! warning "2極法の注意"
    2極法は**スクリーニング用途**として使用する。  
    判定が境界値に近い場合は必ず3極法で再測定すること。

## 測定記録

測定後は以下の項目を記録する：

| 記録項目 | 記入内容 |
|---------|---------|
| 測定日時 | YYYY-MM-DD HH:MM |
| 天候・気温 | 晴れ/曇り/雨、○℃ |
| 接地種別 | A/B/C/D種 |
| 設備名・接地極番号 | 例：No.3変圧器 A種接地 |
| 測定値（3回） | 〇.〇 / 〇.〇 / 〇.〇 Ω |
| 平均値 | 〇.〇 Ω |
| 判定 | OK / NG |
| 測定者・確認者 | 氏名・サイン |

## 判定がNGだった場合の対処

!!! danger "NG判定時の対応"
    判定NGは設備の安全機能低下を意味する。速やかに電計チームリーダーに報告すること。

| 対処方法 | 内容 |
|---------|------|
| 接地線の増設 | 接地極の追加打ち込みまたは接地銅板の埋設 |
| 接地線の点検 | 断線・腐食の確認と交換 |
| 接地端子の清掃 | 端子部の錆・汚損除去と再接続 |
| 大地抵抗率の確認 | 土壌改良（良導電性材料の混合）が必要な場合は設計部門相談 |

## 測定時期の目安

| 接地種別 | 推奨測定頻度 | 法定根拠 |
|---------|------------|---------|
| A種（高圧） | 年1回（年次点検時） | 電気設備技術基準 |
| B種（変圧器中性点） | 年1回（年次点検時） | 電気設備技術基準 |
| C・D種（低圧機器） | 3年に1回以上 | 自主保安計画 |
| 避雷針接地 | 年1回（雷シーズン前） | 建築基準法施行令 |

---

## よくある測定誤差の原因

| 原因 | 対処 |
|------|------|
| 雨天直後（土壌水分過多） | 晴天2日以上後に再測定 |
| 迷走電流の影響 | 測定値が安定しない→夜間または電流源を切断して測定 |
| 補助極が近すぎる | 配置を見直してE〜C間を広げる |
| 接地線を切り離していない | 必ず切り離してから測定 |
| 接地抵抗計の電池切れ | バッテリーチェック機能で確認 |
