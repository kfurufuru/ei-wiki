# 高調波対策

**対象読者**: :material-account-hard-hat: 電気主任技術者 / 電気担当

---

## 30秒まとめ

> 高調波（Harmonics）はインバータ・整流器などの非線形負荷が発生させる、基本波（50/60 Hz）の整数倍の電流です。コンデンサ・リアクトルの過熱、継電器の誤動作、変圧器・中性線の過熱を引き起こします。高圧・特別高圧の需要家は、経済産業省「高調波抑制対策ガイドライン」により受電点から系統へ流出する高調波電流に上限が課され、機器の新設・増設時に流出電流計算での確認が求められます。対策は発生源での抑制（リアクトル・多パルス化）とフィルタ（パッシブ・アクティブ）に大別されます。

このページは高調波の**正典ハブ**です。個別の症状・対策の詳細は各ページへリンクします。

## 高調波の発生源と次数

### 発生源

高調波は電流波形を歪ませる**非線形負荷（Non-linear Load）**から発生します。化学プラントで代表的なものは次のとおりです。

| 発生源 | 主な高調波次数 | 備考 |
|--------|--------------|------|
| インバータ（VFD、6パルス整流入力） | 5・7・11・13次 | 最も一般的な発生源 |
| 整流器・直流電源装置 | 5・7・11・13次 | メッキ電源・充電器など |
| UPS（無停電電源装置） | 5・7次中心 | 入力整流部が発生源 |
| アーク炉・溶接機 | 広帯域・非整数次を含む | 変動が大きく間欠的 |
| 進相コンデンサ（直列リアクトルなし） | 発生源ではないが高調波を増幅 | 共振で被害を拡大 |

### 次数の考え方（n = 6k ± 1）

三相ブリッジ6パルス整流回路が発生させる高調波の次数 n は、理論上次の式で表されます。

```text
n = 6k ± 1   （k = 1, 2, 3, …）

k=1 → 5次・7次
k=2 → 11次・13次
k=3 → 17次・19次
```

- 三相平衡回路では3の倍数次（3・9・15次…）は原理的に打ち消され、系統には流出しにくい
- 12パルス化すると n = 12k ± 1 となり、5・7次が理論上消え11・13次から始まる
- 単相負荷や不平衡がある場合は3次（第3高調波）が中性線に集中する（後述）

!!! note "次数が上がるほど振幅は小さい"
    一般に高調波電流の大きさは次数の逆数に概ね反比例し、5次が最大です。対策は低次（5・7次）の抑制が中心になります。

## 高調波が引き起こす障害

| 被害を受ける設備 | 障害の内容 | 詳細ページ |
|----------------|-----------|-----------|
| 進相コンデンサ | 高調波電流の集中による過熱・膨れ・最悪は破裂 | [力率改善](power-factor.md) |
| 直列リアクトル | 高調波電流による異音（うなり）・過熱・焼損 | [力率改善](power-factor.md) |
| 保護継電器（OCR/GR） | 波形歪みによる不要動作（誤トリップ） | [保護装置誤動作](../06-trouble/protection.md) |
| 零相変流器（ZCT）・地絡継電器 | インバータ高調波・漏洩電流による誤検知 | [保護装置誤動作](../06-trouble/protection.md) |
| 変圧器 | うず電流損の増加による過熱・効率低下 | — |
| 中性線（三相4線式） | 第3高調波（零相）が中性線に重畳し過熱 | — |
| 計装・制御機器 | ノイズ経路での誤動作 | [接地・ノイズトラブル](../06-trouble/grounding-noise.md) |
| 電力量計・力率計 | 波形歪みによる計測誤差 | — |

!!! danger "コンデンサ・リアクトルが「うなる／熱い」は高調波の典型症状"
    進相コンデンサ設備からの異音・過熱は、直列リアクトルなしのコンデンサへの高調波集中や、系統の高調波増大が原因のことが多いです。直列リアクトルの要否・容量選定は[力率改善](power-factor.md)を参照してください。

継電器側の切り分け手順（電流波形の高調波分析、ZCT確認など）は[保護装置誤動作](../06-trouble/protection.md)に、シールド・グラウンドループなどノイズ経路の対策は[接地・ノイズトラブル](../06-trouble/grounding-noise.md)にまとめています。

## 高調波抑制対策ガイドラインと限度値

高圧・特別高圧で受電する需要家には、経済産業省「**高圧又は特別高圧で受電する需要家の高調波抑制対策ガイドライン**」（平成6年制定・平成16年改正）が適用されます。技術的な計算方法は「高調波抑制対策技術指針 JEAG 9702」に基づきます。

### 目的と環境目標レベル

ガイドラインは、各需要家が系統へ流出させる高調波電流に上限を設け、系統電圧の歪みを**高調波環境目標レベル**以下に維持することを目的とします。環境目標レベルは総合電圧歪み率（THD）で定められています。

| 系統 | 総合電圧歪み率の環境目標レベル |
|------|------------------------------|
| 6.6 kV 配電系統 | 5 % |
| 特別高圧系統 | 3 % |

### 対策要否の判定（第1ステップ：等価容量）

高調波発生機器の**等価容量**の合計が基準値を超える需要家が対象になります。

| 受電電圧 | 等価容量の基準値 |
|---------|----------------|
| 6.6 kV（高圧） | 50 kVA 超 |
| 22 kV・33 kV | 300 kVA 超 |
| 66 kV 以上 | 2,000 kVA 超 |

等価容量が基準以下なら、原則としてガイドライン上の対策検討は不要です。

### 流出電流の限度値（第2ステップ）

対象需要家は、受電点で系統へ流出する次数別高調波電流が、契約電力相当値1 kW あたりの**上限値**を超えないことを確認します。6.6 kV 受電の上限値は次のとおりです。

| 次数 | 上限値 [mA/kW]（6.6 kV 受電） |
|------|------------------------------|
| 5次 | 3.5 |
| 7次 | 2.5 |
| 11次以上（13・17・19・23次・23次超） | 次数ごとに個別の上限値を規定（要確認） |

!!! warning "11次以上の具体値は一次資料で確認"
    5次 3.5 mA/kW・7次 2.5 mA/kW は複数の技術資料で確認できましたが、11次以上の各上限値は原典（経済産業省ガイドライン／JEAG 9702）での照合が取れていません。実際の計算・届出時は原典または電力会社（各一般送配電事業者）の様式・技術資料で最新値を確認してください。（要確認 出典: 経済産業省 高調波抑制対策ガイドライン／JEAG 9702）

## 流出電流の計算手順

計算は「機器容量を6パルス整流器換算に直す → 等価容量で要否判定 → 次数別の流出電流を限度値と照合」という流れです。数式はユニコード表記で示します。

### ステップ1: 等価容量の算出

各高調波発生機器の定格容量に、回路種別ごとの**換算係数 Ki** を掛けて合計します。

```text
等価容量 P0 = Σ ( Ki × Pi )

Pi : 各高調波発生機器の定格容量 [kVA]
Ki : 機器の回路種別に応じた換算係数
```

代表的な換算係数（三相ブリッジ6パルス変換装置を基準）は次のとおりです。

| 回路種別 | 換算係数 Ki |
|---------|------------|
| 6パルス変換装置・リアクトルなし | 3.4 |
| 6パルス変換装置・交流側リアクトルあり | 1.8 |
| 6パルス変換装置・直流側リアクトルあり | 1.8 |
| 12パルス変換装置 | 0.5 |

!!! note "換算係数の網羅表は原典を参照"
    上表は代表値です。自励式変換装置・アーク炉など他方式の係数はガイドライン／JEAG 9702 の換算係数表で確認してください。（要確認 出典: JEAG 9702）

### ステップ2: 流出電流の算出と照合

```text
機器ごとの n 次発生電流 In = 定格入力電流 × (n次含有率)
受電点 n 次流出電流 = Σ In × (低減係数・分流係数)

判定: 受電点 n 次流出電流 ≤ 上限値[mA/kW] × 契約電力[kW]
```

- 直列リアクトル付き進相コンデンサがあると系統側への流出が低減されます（低減係数：5次 0.7・7次 0.9 など）
- 上限値を超える次数があれば、対策機器（リアクトル・フィルタ）を追加して再計算します

## 対策の比較

発生源での抑制を優先し、それでも限度を超える場合にフィルタを検討するのが基本の順序です。

| 対策 | 主な効果 | コスト | 適用の目安 |
|------|---------|-------|-----------|
| 交流リアクトル（ACL） | インバータ入力の5・7次を低減 | 低 | 個別インバータへの標準対策 |
| 直流リアクトル（DCL） | ACLより同容量で高調波低減・力率改善が大きい | 低 | インバータ内蔵/後付けの定番 |
| 直列リアクトル（進相コンデンサ用 6%/13%） | コンデンサ保護＋系統高調波の増幅防止 | 低 | コンデンサ設備には必須（詳細は[力率改善](power-factor.md)） |
| 多パルス化（12/18パルス） | 位相シフトで5・7次を打ち消す | 中〜高 | 中〜大容量の変換装置・新設時 |
| パッシブフィルタ（同調フィルタ） | 特定次数（5次等）を低インピーダンスでバイパス | 中 | 特定次数が卓越する系統 |
| アクティブフィルタ | 逆位相電流を注入し広帯域・変動負荷を補償 | 高 | 複数次・変動負荷・既設への後付け |

!!! tip "個別対策が全体対策より安いことが多い"
    発生源（各インバータ）にDC/ACリアクトルを付ける個別対策は、受電点に大型フィルタを据える全体対策より安価で確実なことが多いです。まず発生源対策で流出電流計算が通るかを検討します。

!!! warning "パッシブフィルタと進相コンデンサの共振に注意"
    同調フィルタや進相コンデンサは、系統インピーダンスとの組み合わせで特定次数に共振し、かえって高調波を増幅することがあります。設置前に共振点を確認してください。

## 測定と診断

高調波の実態把握には**高調波測定機能付きのクランプ式電力計・電源品質アナライザ**を使います。

### 見るべき指標

| 指標 | 内容 | 見方 |
|------|------|------|
| THD（電流）| 電流総合歪み率。基本波に対する全高調波の実効値比 | 発生源側の大きさを把握 |
| THD（電圧）| 電圧総合歪み率。環境目標レベル（6.6 kV系 5 %）と比較 | 系統への影響を評価 |
| 次数別スペクトル | 5・7・11・13次…の含有率 | どの次数が卓越するか特定 |

```text
THD = √(I2² + I3² + … + In²) / I1

I1 : 基本波（50/60 Hz）成分の実効値
In : n次高調波成分の実効値
（% 表示は上式を100倍する）
```

- **電圧THD**は系統全体の状態、**電流THD**は個別負荷の発生量を表します。両者を混同しないでください
- 測定は代表的な運転パターン（インバータ高負荷時など）で行い、変動を記録します

## 現場の落とし穴

- **進相コンデンサの共振・進相過剰**: コンデンサ増設や負荷変化で高調波共振や進相過剰が起きることがあります。関係は[力率改善](power-factor.md)を参照してください
- **既設への後付けフィルタ**: 既設系統へフィルタを追加する場合、既存の進相コンデンサとの相互作用（共振）を必ず事前確認します
- **インバータ増設の届出漏れ**: 高調波発生機器の新設・増設は、等価容量が基準を超えると流出電流計算と電力会社への確認が必要です。個別には小さくても合計で基準を超えることがあります
- **3次高調波の中性線集中**: 三相4線式で単相非線形負荷が多いと、第3高調波（零相）が中性線に重畳して過熱します。中性線を相線と同等以上のサイズにするなどの配慮が必要です
