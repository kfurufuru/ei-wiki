# 法定業務一覧

## 電気主任技術者の法的根拠

**電気事業法第43条**: 自家用電気工作物を設置する者は、電気主任技術者を選任しなければならない。

```
自家用電気工作物:
  電圧600V超で受電する需要設備
  （鶴見工場は特別高圧または高圧受電のため対象）
```

電気主任技術者は**保安監督**の責任者であり、設備の工事・維持・運用に関する保安の監督を行う。

> **保安監督とは**: 保安のために必要な指示を行う権限と責任を持つこと。現場作業の直接実施よりも「監督・指示」が本来の役割。

---

## 自家用電気工作物の保安監督義務

### 監督対象設備（鶴見工場の場合）

| 設備種別 | 概要 |
|---------|------|
| 受変電設備 | 特高/高圧受電設備、主変圧器、遮断器（VCB等） |
| 配電設備 | 高圧配電盤、低圧幹線、分岐回路 |
| 電力用機器 | 高圧電動機（6kV等）、コンデンサ、リアクトル |
| 非常用電源 | 非常用発電機、UPS |
| 電気工作物に付属する計装設備 | 保護継電器、電力量計 |

### 保安監督の具体的な義務

1. 月次・年次の法定点検の実施・確認
2. 電気工作物の工事・維持・運用の監督
3. 異常・事故時の応急処置と報告
4. 保安規程の遵守・見直し
5. 従業員への保安教育の実施

---

## 月次・年次の法定点検と記録保管義務

### 月次点検（毎月実施）

| 点検項目 | 確認内容 |
|---------|---------|
| 外観点検 | 異音・異臭・変色・汚損・損傷の確認 |
| 測定記録 | 電圧・電流・電力の指示値確認 |
| 計器確認 | 電力量計の正常動作、保護継電器の表示確認 |
| 開閉器動作確認 | LBS・DS等の動作状態確認 |
| 接地線の確認 | 断線・腐食がないか |

**記録保管**: 月次点検記録は**最低3年間**保管（保安規程で定める）

### 年次点検（停電を伴う精密点検）

| 点検項目 | 内容 | 実施頻度 |
|---------|------|---------|
| 絶縁抵抗測定 | 電路・機器の絶縁抵抗（MΩ測定） | 年1回以上 |
| 絶縁耐力試験 | 規定電圧での耐力確認（新設・大修繕後） | 工事後 |
| 保護継電器試験 | 動作電流・時間の整定値確認 | 年1〜3回（保安規程による） |
| 変圧器点検 | 絶縁油採取・試験、タップ確認 | 年1回以上 |
| 遮断器点検 | 動作確認、接触抵抗測定、操作回路確認 | 年1回以上 |
| 接地抵抗測定 | A種・B種・C種・D種各接地の抵抗値 | 年1回以上 |

!!! warning "年次点検の記録は法定保管"
    年次点検の記録は電気事業法施行規則により保管が義務付けられる。
    記録内容・形式は保安規程で定め、実態と合致させること。

## 電気事故報告の義務

電気事業法第106条・施行規則第99条に基づき、一定規模以上の電気事故は経済産業大臣へ報告義務がある。

### 報告が必要な事故の範囲

| 事故種別 | 報告要否 |
|---------|---------|
| 感電による死傷事故（死亡・入院） | **要報告** |
| 電気火災（建物等に延焼した場合） | **要報告** |
| 主要電気工作物（変圧器・遮断器等）の損傷 | **要報告**（電圧・規模による） |
| 特別高圧設備の停電事故 | **要報告** |
| 構内事故（軽微・短時間） | 要否は個別判断 |
| 軽微なヒューズ切れ・漏電遮断器作動 | **不要**（記録は残す） |

> **判断に迷ったら**: 所轄の産業保安監督部（関東では関東東北産業保安監督部）に事前確認することを推奨。報告基準の解釈を口頭確認しておくと事故時の対応が速い。

### 報告期限

| 報告種別 | 提出期限 |
|---------|---------|
| 速報 | 事故発生から**24時間以内** |
| 確報 | 事故発生から**30日以内** |

詳細な対応手順は [jiko-taiou.md](jiko-taiou.md) を参照。

## 保安規程の作成・変更届出

- 自家用電気工作物の設置者は**保安規程を作成**し、使用開始前に経済産業大臣へ届出が必要
- 変更時も届出が必要（軽微変更を除く）

詳細は [hoan-kisoku.md](hoan-kisoku.md) を参照。

---

## 工事計画届出が必要なケース

電気事業法第48条に基づき、一定規模以上の工事は事前に届出（または認可）が必要。

### 届出が必要な主なケース

| 工事内容 | 届出要否 | 届出先 |
|---------|---------|--------|
| 受電設備の**最大電力500kW以上**の増設・変更 | **要届出** | 産業保安監督部 |
| 特別高圧変圧器（10,000kVA以上）の設置・変更 | **要届出** | 産業保安監督部 |
| 高圧遮断器の新設（遮断容量変更を伴う場合） | ケースによる | 要確認 |
| 非常用発電機（10kW以上）の新設 | **要届出** | 産業保安監督部 |
| 低圧設備のみの改造（受電点に影響なし） | **不要** | — |

### 届出のタイミング

```
工事計画届出の提出
    ↓
受理後、30日間（軽微は15日間）待機
    ↓（届出受理後、法定待機期間経過後）
工事着工
    ↓
竣工検査（保安監督）
    ↓
使用前安全管理検査（大規模設備の場合）
    ↓
使用開始
```

!!! danger "届出なしで着工した場合"
    工事計画届出が必要な工事を届出なしで行った場合、電気事業法違反となる（100万円以下の罰金）。
    設備改造の計画段階で電気主任技術者が届出要否を確認することが重要。
