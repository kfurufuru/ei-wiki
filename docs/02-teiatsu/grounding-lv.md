# 接地（低圧）

## 30秒まとめ

低圧接地の基本は「D 種接地（10Ω以下）で機器を保護接地する」こと。計装信号のシグナルグランドは保護接地と分離して一点接地にする。インバータのノイズ接地はシールド線と本体を同一点に落とし、ループアースを作らない。

---

## D 種接地（第 3 種接地工事）

| 項目 | 内容 |
|------|------|
| 接地抵抗値 | 10Ω 以下（300V 以下の低圧機器） |
| 対象機器 | 400V 以下の電動機外箱・制御盤・変圧器二次側 |
| 接地線断面積 | 2.5mm² 以上（電技解釈第 17 条） |
| 色規格 | 緑または緑/黄縞（JIS） |

!!! info "C 種接地との違い"
    C 種（第 2 種相当）は 10Ω 以下で、高圧機器が低圧側に混触した場合の危険電圧を抑制するために使用。低圧のみの設備では D 種が標準。

---

## シグナルグランドと保護接地の分離原則

### 混触リスク

保護接地（PE）は漏電電流・ノイズ電流が流れる。計装信号のシグナルグランド（SG）と同一接地にすると、ノイズが信号に乗り込む。

```
悪い例（PE と SG を共用接地）:
    センサー SG ─────┬── 共用接地 ── 大地
                     │
    インバータ PE ───┘
    （インバータのスイッチングノイズがセンサー信号に干渉）

良い例（分離接地）:
    センサー SG ──── 計装用接地 ──── 大地  ┐
                                            ├ 等電位ボンディング（低インピーダンス）
    インバータ PE ── 保護接地 ──── 大地    ┘
```

### 一点接地の原則

シグナルグランドは計装盤内の一点（スター接地点）に集め、そこから 1 本で大地に落とす。複数点で接地するとループが形成され誘導ノイズの原因になる。

---

## インバータのノイズ接地（EMC 対策）

| 対策 | 内容 |
|------|------|
| インバータ本体 PE | D 種接地極に直結（専用線、最短経路） |
| モーターケーブル シールド | 盤側端のみ接地（両端接地はループアース）または両端接地（EMC フィルタと組み合わせ） |
| EMC フィルタ接地 | インバータ一次側と同一接地点に接続 |
| 計装ケーブルとの分離 | インバータ配線と計装信号線を 300mm 以上離隔 |

!!! warning "シールドの接地方法"
    シールドケーブルの両端接地はループアンテナになりノイズを拾う場合がある。原則は盤側の一端接地とし、長距離でノイズが乗る場合は両端接地＋EMC フィルタの組み合わせを検討する。

## 等電位ボンディング

### 目的

1. **感電防止**：人体が触れる金属部分の電位差をなくす
2. **爆発防止**：静電気の放電スパークを防ぐ（化学プラント）

### 対象

- 金属製ケーブルラック
- 金属製配管（工程ライン・ユーティリティ配管）
- プラント構造物の金属フレーム
- 機器の金属外箱

```
接地銅バー（等電位ボンディングバー）
    │
    ├── ケーブルラック（ボンディング線 22mm² 以上）
    ├── 配管支持金物（ボンディング線 14mm² 以上）
    ├── 操作盤外箱
    └── D 種接地極
```

---

## 接地線の色規格と断面積選定

| 用途 | 色 | 最小断面積 |
|------|----|---------|
| 保護接地線（PE） | 緑または緑/黄縞 | 2.5mm²（相線 16mm² 以下の場合） |
| 中性線（N） | 白または水色 | 相線と同断面積 |
| 等電位ボンディング線 | 緑または緑/黄縞 | 2.5mm² 以上（電技解釈） |
| 計装シグナルグランド | 緑または裸線 | 2mm² 以上 |

!!! danger "緑の電線は接地専用"
    緑色の電線を接地以外の配線に使用してはならない（電技解釈・内線規程）。誤って信号線や制御線に使うと保守作業時の感電事故につながる。
