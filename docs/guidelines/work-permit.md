# 作業許可証（PTW）発行手順

Permit to Work（PTW / 作業許可証）の申請・発行・作業完了の手順です。  
工場内のすべての電気計装作業は PTW の取得が必要です。

---

!!! danger "PTW なしの作業は禁止"
    作業許可証を取得していない状態での設備作業は、社内規程違反です。  
    「ちょっとだけ」「急いでいるから」は理由になりません。

---

## PTW の目的

1. **危険エネルギーの管理**：電気・圧力・熱エネルギーの隔離・除去を確実に行う
2. **関係者への情報共有**：製造課・保全・安全部門が同一の認識を持つ
3. **責任の明確化**：誰が作業し、誰が承認したかを記録する
4. **再確認の仕組み**：作業前のリスクアセスメントを強制する

---

## PTW の種類（工場によって異なる場合がある）

| PTW 種別 | 対象作業 |
|---------|---------|
| 一般作業許可 | 通常の保全・点検作業 |
| 熱作業許可（Hot Work） | 溶接・切断・グラインダー等の火花発生作業 |
| 密閉空間入域許可 | ピット・タンク・マンホール内作業 |
| 高所作業許可 | 高さ2m以上の作業 |
| 特別高・高圧電気作業許可 | 高圧活線近接・停電作業 |

---

## PTW 発行の全体フロー

```
作業計画立案 → PTW 申請（作業者） → リスクアセスメント実施
→ 製造課承認（運転員・班長） → 安全部門確認（必要な場合）
→ 現場の危険エネルギー隔離 → 作業開始 → 作業完了
→ 現場復旧確認 → PTW クローズ
```

---

## STEP 1：作業計画の立案（作業前日まで）

1. 以下の内容を整理する
    - **作業場所**：建屋名・設備名・タグ番号
    - **作業内容**：具体的な作業（例：インバータ点検・配線変更・計器校正）
    - **作業時間**：開始〜終了の予定時刻
    - **作業者**：氏名・資格（電気工事士・特別教育等）
    - **必要な安全措置**：停電・ロックアウト・立入禁止区画等

2. 作業対象設備の**P&ID・回路図・単線結線図**を確認する
3. 前回のトラブル・注意事項を過去記録で確認する

!!! info "複数日にわたる作業"
    複数日にわたる場合は**作業日ごとにPTWを更新**する。  
    前日のPTWを翌日も使いまわすことは禁止。

---

## STEP 2：PTW 申請書の記入

所定の PTW 申請書（紙 or 電子システム）に記入する。

### 必須記入項目

| 項目 | 記入内容 |
|------|---------|
| 申請日時 | YYYY-MM-DD HH:MM |
| 作業場所 | 詳細な場所（棟・エリア・設備番号） |
| 作業内容 | 具体的・明確に（「点検」だけでは不可） |
| 作業者 | 全員の氏名・資格（リーダーを明記） |
| 予定時間 | 開始〜終了 |
| 危険エネルギー | 電気・熱・圧力・化学物質等 |
| 隔離方法 | 停電・ロックアウト・バルブ閉止等 |
| 必要な保護具 | 絶縁手袋・保護メガネ・難燃服等 |
| 緊急時連絡先 | 現場最寄りの内線番号 |

### リスクアセスメント記入

PTW 申請書のリスク欄に以下を記入する：

| 危険要因 | 最悪のケース | リスクレベル | 対策 |
|---------|------------|------------|------|
| 感電 | 死亡・重傷 | 高 | 停電・検電・絶縁保護具 |
| 高所転落 | 死亡・重傷 | 高 | 安全帯・手摺り・落下防止 |
| 工具の落下 | 下方作業者の怪我 | 中 | 工具係留・立入禁止区画 |

---

## STEP 3：製造課への申請・承認

1. 作業予定日の**前日まで**に製造課（担当班長・当直長）に PTW を持参して説明する
2. 以下の内容を口頭で説明する
    - 作業の目的・内容
    - 停止が必要な設備と**停止時間の見込み**
    - 生産への影響（停止・減産・代替手段）
3. 製造課の承認サインをもらう

!!! warning "製造課の承認なしに作業開始禁止"
    「口頭で言った」だけでは承認にならない。  
    必ずサインをもらい、両者の手元に控えが残るようにする。

---

## STEP 4：危険エネルギーの隔離（LOTO）

作業直前に、製造課立会いのもとで実施する。

### 電気的隔離（ロックアウト）手順

1. 対象回路の**ブレーカ・断路器を開放（OFF）**する
2. **ロック錠**を取り付ける（作業者ごとに専用の錠と鍵を使用）
3. **タグ（Do Not Operate タグ）**を取り付ける

```
ロック錠 → 作業者が鍵を保持している間は誰も投入できない仕組み
```

4. **検電器**で確実に停電を確認する
    - 三相全相・各相-接地間を確認
    - 検電器自体の動作確認を事前に行う
5. **接地棒**を取り付ける（誘導電圧・誤通電対策）

!!! info "ロックアウト・タグアウト（LOTO）"
    ロックアウトは「物理的に操作できなくする」。  
    タグアウトは「操作しないよう警告する」。  
    両方を実施することが原則。タグアウトのみは許可されていない。

### 複数の隔離点がある場合

- 複数のブレーカ・バルブを隔離する場合は**すべての隔離点を記録**する
- PTW の隔離点リストに全点を記載する
- 隔離点の見落としが感電事故の最大の原因

---

## STEP 5：作業開始

1. 全ての安全措置が完了したことを確認する
2. 作業者全員に PTW の内容を周知する（ツールボックスミーティング実施）
3. **作業開始時刻を PTW に記録する**
4. 作業中は PTW を現場に携帯（または現場に掲示）する

### ツールボックスミーティング（TBM）のチェックポイント

- [ ] 本日の作業内容の確認
- [ ] 危険箇所の確認（隔離完了確認）
- [ ] 保護具の着用確認
- [ ] 緊急連絡先の確認
- [ ] 体調の確認（「体調不良者は申し出てください」）

---

## STEP 6：作業中の管理

1. 作業内容が PTW の範囲を超えそうな場合は**作業を止めて PTW を再申請**する
2. 隔離した設備に**他の人が触らないよう**監視する
3. 作業範囲外への立入を防ぐ区画（バリケード・カラーコーン）を設置する
4. 問題が発生した場合は PTW 申請者（リーダー）に即報告する

---

## STEP 7：作業完了・PTW クローズ

### 現場の確認

1. 工具・材料・廃材をすべて回収する（**現場の7S確認**）
2. 盤扉・カバー・安全カバーをすべて復旧する
3. 接地棒を取り外す
4. ロック錠・タグを取り外す（**作業者本人のみが取り外す**）

!!! danger "他人のロックを外すことは絶対禁止"
    他の作業者のロック錠を無断で取り外すことは、その作業者の安全を脅かす行為。  
    例外なく禁止。鍵をなくした・作業者が帰った等の場合は手続きに従う。

5. ブレーカ・断路器を復旧する前に**製造課に連絡する**

### 製造課への完了連絡

1. 「作業が完了した」「安全措置を解除する」ことを製造課に連絡する
2. 製造課の立会いのもとで電源を投入する（または製造課が投入する）
3. 設備の動作確認を実施する

### PTW クローズ

1. PTW 完了欄に**完了時刻・完了サイン**を記入する
2. 製造課担当者のサインをもらう
3. PTW を**所定の場所に保管**する（保管期間：通常1年以上）

---

## チェックリスト（PTW クローズ時）

- [ ] 全工具・材料の回収完了
- [ ] 盤・設備カバーの復旧完了
- [ ] 接地棒の取り外し完了
- [ ] ロック錠・タグの取り外し完了（作業者本人が実施）
- [ ] 製造課への完了連絡済み
- [ ] 電源復旧・動作確認完了
- [ ] PTW 完了サイン・保管完了

---

## よくある PTW ミスと防止策

| ミス | 防止策 |
|------|--------|
| 前日の PTW を使いまわす | 毎日更新・日付確認 |
| 隔離点の漏れ | 隔離点リストをすべて記入してから作業開始 |
| 作業範囲の拡大（クリープ） | 範囲を超えそうになったら即停止・再申請 |
| 他人のロックを外す | 手続き（上長承認）なしで外さない |
| 完了報告なしに作業終了 | 製造課への連絡を手順に組み込む |
