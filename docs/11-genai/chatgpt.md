# ChatGPT活用

## ChatGPTとは

OpenAI社が開発した対話型AIサービス。ブラウザ（chat.openai.com）またはスマホアプリから利用できる。

テキストを入力（チャット）するだけで、質問への回答・文書作成・翻訳・要約など幅広いタスクをこなす。

---

## 無料版と有料版の違い

| 項目 | 無料版（Free） | 有料版（Plus/月$20） |
|------|--------------|-------------------|
| 使えるモデル | GPT-4o mini（軽量版） | GPT-4o（高性能版） |
| 利用制限 | あり（長時間使用で制限） | 優先アクセス |
| 画像入力 | 制限あり | 利用可 |
| データ分析 | 非対応 | 対応 |
| おすすめ対象 | 試してみたい人・軽い用途 | 毎日業務で使う人 |

!!! tip "まずは無料版で試す"
    無料版でも基本的な文書作成・翻訳・Q&Aは十分使える。毎日使いたいと感じたら有料版を検討。

---

## 基本操作

1. `chat.openai.com` にアクセス（またはアプリを起動）
2. テキストボックスに指示を入力して送信
3. 回答が表示される
4. 不満足な場合は「もっと具体的に」「箇条書きで」「例を追加して」と追加で依頼
5. 新しい話題は「New chat」で新しい会話を始める（会話が長くなると精度が落ちる場合がある）

## 電気計装エンジニア向け業務活用例

| 業務 | 活用例 | プロンプトのポイント |
|------|--------|-------------------|
| 技術調査 | HARTプロトコルの仕組みを知りたい | 「電気計装エンジニア向けに」と役割を指定する |
| 文書作成 | 計装仕様書・電気仕様書のドラフト | 仕様の条件を箇条書きで渡す |
| メール作成 | 社外向けメールの文体を整える | 相手・目的・トーンを指定する |
| トラブル解析 | OCトリップの原因候補を列挙させる | 症状・設備情報・確認済みの内容を伝える |
| 翻訳 | 英語仕様書・IEC規格の日本語訳 | 「技術翻訳として」と指定すると専門用語に配慮してくれる |
| アイデア出し | 省エネ対策・改善アイデアの列挙 | 制約条件（コスト・設備規模等）を先に伝える |

## 業務別プロンプト例

### 技術調査

```
電気計装エンジニア向けに、HARTプロトコルの以下を説明して：
1. 基本的な仕組み（4-20mAとの関係）
2. HART対応機器で何ができるか
3. 現場での活用場面

専門用語を使ってよいが、各項目は3〜5行程度で簡潔に。
```

### トラブル解析

```
電気計装エンジニアとして回答して。

症状：圧力伝送器（4-20mA、2線式）の出力が急に4mAになった。
設備：DCS側で断線アラーム発報。現場でループ電源確認済み（24VDC正常）。
試したこと：端子台の接触確認済み（異常なし）。

原因候補を確率が高い順に5つ挙げて。各原因に対して現場での確認手順も教えて。
```

### 仕様書ドラフト

```
以下の条件で電動バルブの計装仕様書（概要版）を作成して：
- バルブ種別：電動ゲートバルブ
- 配管口径：4インチ（100A）
- プロセス流体：温水（80℃以下）
- 制御方式：DCSからの開閉指令（DO出力）
- 開閉確認：リミットスイッチによるDI入力
- 電源：AC100V単相
- 防護等級：IP65以上
- 記載項目：概要・電気仕様・信号仕様・設置条件

項目ごとに整理して、A4 1枚程度の内容で。
```

### メール作成（社外向け）

```
以下の内容でメーカーへの問い合わせメールを作成して：
・件名：〇〇（型番）の予備品の在庫確認
・確認したい内容：部品A（品番：XXXXX）の現在の在庫状況と納期
・急いでいる理由：設備の予防保全で翌月に使用予定
・文体：丁寧なビジネスメール、件名〜署名まで

社名・個人名は「〇〇」のままにしておいて。
```

### 英語資料の翻訳

```
以下の英文技術仕様を日本語に翻訳して。
電気計装の専門用語は適切な日本語の業界用語に変換すること。

[英文を貼り付け]
```

## Claudeとの使い分け

| 用途 | 向いているツール | 理由 |
|------|--------------|------|
| 長文ドキュメントの作成・読解 | Claude | 長文の理解・生成が得意 |
| 短い質問・素早いQ&A | ChatGPT | レスポンスが速い |
| Web検索が必要な調査 | Bing Copilot | リアルタイム情報に対応 |
| Officeツールと連携した作業 | M365 Copilot | Office統合が強み |
| コード作成・データ分析 | ChatGPT（有料版） | Code Interpreterで実行も可 |

## 注意事項

!!! danger "社外秘情報の入力は禁止"
    ChatGPT・Claude.aiは個人向けサービスで、入力内容がOpenAI/Anthropicのサーバーに送信される。
    **設備の詳細情報・プロセス条件・社外秘の業務情報は絶対に入力しない。**
    詳細は[注意事項・情報セキュリティ](security.md)を参照。
