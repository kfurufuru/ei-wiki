# トラブル逆引きインデックス

> 症状から原因へ — まず症状を選んでください

対象: 電気担当 / 保全担当 / 製造担当

## 症状から探す

| 症状 | 最初に確認すること | 詳細記事 |
|------|------------------|----------|
| ブレーカーが落ちる / 遮断器がトリップ | 過負荷か短絡か地絡かを判別 | [電源トラブル](power.md) |
| モーターが起動しない | 電源・制御回路・機械的ロックを確認 | [モータートラブル](motor.md) |
| モーターが過熱する | 過負荷・通風・絶縁劣化を確認 | [モータートラブル](motor.md) |
| インバータがエラー停止 | エラーコードを確認 | [インバータトラブル](inverter.md) |
| 計装信号が異常値 / ゼロ貼り | 断線・電源・センサ本体を確認 | [計装信号トラブル](signal.md) |
| 制御弁が動かない / 全開全閉 | ポジショナ・空気圧・フェイルセーフを確認 | [制御弁トラブル](valve.md) |
| PLC / DCS がアラーム | アラームコードとI/Oステータスを確認 | [PLC/DCSトラブル](plc-dcs.md) |
| ノイズが乗る / 信号が不安定 | アース・シールド・配線ルートを確認 | [接地・ノイズ](grounding-noise.md) |
| 保護継電器が誤動作 | 整定値・CT二次側・外部要因を確認 | [保護装置誤動作](protection.md) |
