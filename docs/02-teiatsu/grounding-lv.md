# 接地（低圧）

## 30秒まとめ

低圧接地の基本は「D 種接地（100Ω以下）で機器を保護接地する」こと。計装信号のシグナルグランドは保護接地と分離して一点接地にする。インバータのノイズ接地はシールド線と本体を同一点に落とし、ループアースを作らない。

!!! info "用語・分類の整理は計装側へ"
    GND と接地の **違いと種類（FG/SG/AGND/DGND/PGND）の定義** は [グランドと接地](../03-keiso/grounding-gnd.md) に整理しています。
    本ページは **低圧の接地工事（D 種など）の実装** を扱います。

## D 種接地（第 3 種接地工事）

| 項目 | 内容 |
|------|------|
| 接地抵抗値 | 100Ω 以下（地絡時 0.5 秒以内に自動遮断する装置を施設する場合は 500Ω 以下） |
| 対象機器 | 300V 以下の低圧機器（電動機外箱・制御盤・変圧器二次側等） |
| 接地線断面積 | 2.5mm² 以上（電技解釈第 17 条） |
| 色規格 | 緑または緑/黄縞（JIS） |

!!! info "C 種接地との違い"
    C 種（旧・特別第 3 種接地工事）は **300V を超える低圧機器の保護接地** に使用し、10Ω 以下（0.5 秒以内に自動遮断する装置を施設する場合は 500Ω 以下）とします。D 種は 300V 以下の低圧機器が対象で、両者は「対象機器の電圧区分」で使い分けます。<!-- lint-ok: N1 C種の10Ωは正・D種との対比行 -->
    なお、高低圧混触時に低圧側の電位上昇を抑制する接地は **B 種接地**（変圧器低圧側中性点）の役割であり、C 種・D 種とは目的が異なります。<!-- lint-ok: N2 混触抑制はB種の役割と明記する正当行 -->

## シグナルグランドと保護接地の分離原則

### 混触リスク

保護接地（PE）は漏電電流・ノイズ電流が流れる。計装信号のシグナルグランド（SG）と同一接地にすると、ノイズが信号に乗り込む。

```
悪い例（PE と SG を共用接地）:
    センサー SG ─────┬── 共用接地 ── 大地
                     │
    インバータ PE ───┘
    （インバータのスイッチングノイズがセンサー信号に干渉）

良い例（分離接地）:
    センサー SG ──── 計装用接地 ──── 大地  ┐
                                            ├ 等電位ボンディング（低インピーダンス）
    インバータ PE ── 保護接地 ──── 大地    ┘
```

### 一点接地の原則

シグナルグランドは計装盤内の一点（スター接地点）に集め、そこから 1 本で大地に落とす。複数点で接地するとループが形成され誘導ノイズの原因になる。

## インバータのノイズ接地（EMC 対策）

| 対策 | 内容 |
|------|------|
| インバータ本体 PE | D 種接地極に直結（専用線、最短経路） |
| モーターケーブル シールド | 盤側端のみ接地（両端接地はループアース）または両端接地（EMC フィルタと組み合わせ） |
| EMC フィルタ接地 | インバータ一次側と同一接地点に接続 |
| 計装ケーブルとの分離 | インバータ配線と計装信号線を 300mm 以上離隔 |

!!! warning "シールドの接地方法"
    シールドケーブルの両端接地はループアンテナになりノイズを拾う場合がある。原則は盤側の一端接地とし、長距離でノイズが乗る場合は両端接地＋EMC フィルタの組み合わせを検討する。

## 等電位ボンディング

### 目的

1. **感電防止**：人体が触れる金属部分の電位差をなくす
2. **爆発防止**：静電気の放電スパークを防ぐ（化学プラント）

### 対象

- 金属製ケーブルラック
- 金属製配管（工程ライン・ユーティリティ配管）
- プラント構造物の金属フレーム
- 機器の金属外箱

```
接地銅バー（等電位ボンディングバー）
    │
    ├── ケーブルラック（ボンディング線 22mm² 以上）
    ├── 配管支持金物（ボンディング線 14mm² 以上）
    ├── 操作盤外箱
    └── D 種接地極
```

## 接地線の色規格と断面積選定

| 用途 | 色 | 最小断面積 |
|------|----|---------|
| 保護接地線（PE） | 緑または緑/黄縞 | 2.5mm²（相線 16mm² 以下の場合） |
| 中性線（N） | 白または水色 | 相線と同断面積 |
| 等電位ボンディング線 | 緑または緑/黄縞 | 2.5mm² 以上（電技解釈） |
| 計装シグナルグランド | 緑または裸線 | 2mm² 以上 |

!!! danger "緑の電線は接地専用"
    緑色の電線を接地以外の配線に使用してはならない（電技解釈・内線規程）。誤って信号線や制御線に使うと保守作業時の感電事故につながる。
