## 使い方のヒント

- 画面上部の検索ボックスは全ページ横断で見出し・本文を検索します。略語（CB / MCCB / OCR など）でも引けます。
- 用語集に登録された略語は `abbreviations.md` 経由で全ページに自動でツールチップ付与されます。本文中の略語にマウスを乗せると定義が出ます。
- 計算ツールはブラウザに保存（ブックマーク）しておけば、オフライン環境でも式と早見表を参照できます。
- 各記事のフッタ「🤖 ChatGPTに渡す」ボタンで、記事URL付きの質問プロンプトを生成できます。
- 監査バッジ（最終確認日）が橙・赤の記事は、参照前に最新規格・法令を確認してください。
