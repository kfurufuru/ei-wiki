# 規格・法規

## 30秒まとめ

化学プラントの電気計装に関わる法規は「電気事業法・高圧ガス保安法・労働安全衛生法・消防法」の4本柱。法規を先に押さえて設計を進める。後から法規制に気づくと設計変更・届出遅延が発生する。

---

## 化学プラントに特有の法規制一覧

| 法規 | 主管省庁 | 化学プラントでの主な適用 |
|------|---------|----------------------|
| 電気事業法 | 経済産業省 | 自家用電気工作物（600V 超）の設置・変更届出、定期点検義務 |
| 電気設備技術基準（電技） | 経済産業省 | 絶縁・接地・防護の最低基準 |
| 内線規程（JEAC 8001） | 電気技術規程 | 屋内配線の設計・施工基準（事実上の標準） |
| 高圧ガス保安法 | 経済産業省 | 高圧ガス設備の変更届出・完成検査・定期自主検査 |
| 労働安全衛生法 | 厚生労働省 | 停電作業・特別教育・設備の安全基準 |
| 消防法 | 総務省（消防庁） | 防爆機器の設置・危険物取扱・非常用電源 |
| 火薬類取締法 | 経済産業省 | 爆発物を扱う設備（該当する場合） |

---

## 電気事業法・電気設備技術基準

| 適用事項 | 内容 |
|---------|------|
| 自家用電気工作物の定義 | 最大電力 50kW 以上の需要設備（ほぼすべての工場） |
| 主任技術者の選任 | 電気主任技術者（第 1〜3 種）の選任義務 |
| 定期点検 | 年次点検（停電点検）の実施と記録保存 |
| 変更届出 | 電気工作物の変更工事前に届出が必要 |
| 接地の基準 | A〜D 種接地の抵抗値・施工基準（電技解釈第 17 条） |

---

## 高圧ガス保安法

| 適用事項 | 内容 |
|---------|------|
| 設備変更届 | 高圧ガス製造設備の電気・計装系統変更は事前届出 |
| 軽微変更届 | 同等品交換・軽微な変更は軽微変更届で対応 |
| 完成検査 | 変更工事完了後、試運転前に完成検査を受検 |
| 定期自主検査 | 年 1 回以上の定期自主検査と記録保存（内部検査） |
| 保安検査 | 特定施設は 2 年に 1 回の都道府県保安検査 |

!!! warning "電気・計装の変更が法規適用になるケース"
    - 安全装置（安全弁・緊急遮断弁・高圧スイッチ）の制御ロジック変更
    - 計測器の変更による制御精度の変化
    - ポンプ・コンプレッサーの電気制御系変更
    これらは「製造方法の変更」として届出が必要なことがある。事前に保安担当者と確認する。

## 労働安全衛生法

| 適用事項 | 内容 |
|---------|------|
| 特別教育（低圧取扱） | 低圧電気取扱業務の特別教育修了者が作業 |
| 特別教育（高圧・特高） | 高圧・特別高圧電気取扱業務の特別教育 |
| 停電作業の措置 | 停電確認・短絡接地の実施義務（労安則第 339〜341 条） |
| 感電防止措置 | 活線作業の絶縁保護具着用義務 |
| 設備の安全基準 | 危険を防止するための機械設備の安全化義務 |

---

## 消防法（防爆・危険物）

| 適用事項 | 内容 |
|---------|------|
| 危険物施設の電気設備 | 危険物取扱施設の電気設備は防爆型が必要（消防法第 10 条） |
| 非常用電源 | 消防設備への非常用電源（非常用発電機・UPS）の確保義務 |
| 避難誘導灯・非常照明 | 設置基準・バックアップ時間の規定 |
| 防爆機器の届出 | 製造所等変更時に消防署への変更届出が必要な場合あり |
