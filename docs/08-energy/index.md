# 08 省エネ・エネルギー管理

当工場の電気・計装チームが日常的に関わる省エネ・エネルギー管理業務の実務リファレンス。
省エネ法の法的義務から、デマンド管理・インバータ省エネ計算・力率改善ROIまでカバーする。

## セクション概要

| 記事 | ファイル | 対象業務 |

| 省エネ法対応 | [energy-law.md](energy-law.md) | 法定報告・エネルギー管理士選任 |
| デマンド管理 | [demand-management.md](demand-management.md) | デマンド超過防止・負荷制御 |

## このセクションの使い方

- **省エネ法の報告期限が迫っている** → [energy-law.md](energy-law.md) で提出項目を確認
- **デマンド警報が鳴った** → [demand-management.md](demand-management.md) の対応手順へ
- **インバータ導入の稟議を通したい** → [vfd-energy.md](vfd-energy.md) で計算式と計算例を参照
- **力率ペナルティを受けている** → [power-factor-roi.md](power-factor-roi.md) でROI試算
- **エネルギー消費の異常を疑っている** → [energy-monitoring.md](energy-monitoring.md) の監視手法を確認
