# 工事完了後の受入実務

## 30秒まとめ

ベンダーの「工事完了」は **確認開始の合図** であり、完了宣言ではない。  
設備技術部担当者がやるべきことは **現場・性能・書類・引継ぎ・残件管理** の5軸を閉じること。  
「動いた」ではなく「安全に維持管理できる状態になった」が真の完了基準。

---

## 全体フロー

```
ベンダー「工事完了」報告
        ↓
① 完了範囲・変更点・残件の確認
        ↓
② 現地完成検査（施工品質・外観）
        ↓
③ 試運転・性能確認
        ↓
④ 安全・法令・社内基準の適合確認
        ↓
⑤ 完成図書・試験記録の回収
        ↓
⑥ 運転/保全部門への引継ぎ
        ↓
⑦ 残課題の是正管理
        ↓
⑧ 検収判断 → 支払連携
```

---

## ① 完了範囲の確認（最優先）

!!! danger "ここが曖昧だと「そこは対象外」になる"
    完了報告を受けた直後に必ず確認する。後工程の手戻りを防ぐ唯一のタイミング。

| 確認項目 | チェック観点 |
|---------|-----------|
| 施工スコープ | どこまでが今回工事の対象か |
| 残工事・未施工 | 未完了の作業項目がないか |
| 手直し項目 | 指摘事項への対応が完了しているか |
| 変更点 | 当初仕様からの変更・代替品の有無 |

---

## ② 現地完成検査（施工品質・外観）

机上確認だけでは不十分。**必ず現地に足を運ぶ。**

### 外観・施工品質の確認項目

| カテゴリ | 確認内容 |
|---------|---------|
| 施工 | 図面・仕様書どおりに施工されているか |
| 品質 | 溶接・締結・塗装・防食処理に問題がないか |
| ケーブル・配線 | シールド処理・端末処理・マーキングの適否 |
| 支持・固定 | 配管・ケーブルラックの支持間隔・締め付けトルク |
| 表示類 | 銘板・危険表示・ループ番号ラベルの貼付 |
| 養生撤去 | 養生材・廃材・仮設物の撤去、清掃完了 |
| 保全性 | メンテナンスアクセス・スペースの確保 |
| 干渉 | 既設設備との干渉・クリアランス |

!!! warning "「動けばOK」ではない"
    保全性・安全性・変更点を含む総合評価が必要。  
    竣工時に見落とした不備は保全時に事故要因になる。

---

## ③ 試運転・性能確認

設備技術部の責務は「設置された」ではなく **「要求どおり機能する」** まで確認すること。

### 確認ステップ

```
単体確認
  └─ 電源投入・ランプ確認
  └─ ゼロ点・スパン調整
  └─ 入出力信号の導通確認

インターロック確認
  └─ 各インターロック条件でのシャットダウン動作
  └─ バイパス設定の確認

保護動作確認
  └─ 異常警報（Hi/Lo）の発報確認
  └─ 保護リレー動作確認

停止・復旧動作
  └─ 緊急停止・復旧手順の動作確認

実負荷運転
  └─ 設計流量・圧力・温度での安定動作
  └─ PVの追従・制御ハンチングのないこと
  └─ 記録（トレンド・ログ）の取得
```

---

## ④ 安全・法令・社内基準の適合確認

工事完了後に見落としやすいが、後から発覚すると致命的。

| 区分 | 確認内容 |
|-----|---------|
| 感電防止 | 充電部の露出・感電危険箇所の保護カバー |
| 接地 | 接地工事の完了・接地抵抗値の測定記録 |
| 絶縁 | 絶縁抵抗測定記録 |
| LOTO対象 | 新設機器のロックアウト対象リストへの追加 |
| 防爆 | 防爆エリアの防爆機器適合・ケーブルグランド施工 |
| 法令届出 | 電気工作物変更届・消防法等の届出要否の確認 |
| 社内基準 | 設備技術標準・保全基準・管理番号体系への適合 |

!!! note "電気設備は法令適合と記録保全が後から効く"
    接地抵抗・絶縁抵抗の初期値は将来の劣化判断の基準となる。必ず数値記録を残す。

---

## ⑤ 完成図書・試験記録の回収

**ここをやらないと将来の保全で必ず困る。**

### 最低限回収すべき書類

| 書類 | 要点 |
|-----|-----|
| 完了報告書 | 工事内容・変更点の記録 |
| 完成図（As-built） | **現場施工状態を反映した最新図面** |
| 回路図・配線図・配置図 | DCS タグ名・端子番号まで反映済みか確認 |
| ループ図（Loop Diagram） | 端子番号・シールド接地点の記載 |
| 試験成績書・測定記録 | 接地抵抗・絶縁抵抗・ループチェック結果 |
| 取扱説明書・保守要領書 | 機器ごとのメーカー純正マニュアル |
| 使用機器リスト・型式一覧 | 予備品手配・保全の基礎情報 |
| 保証書 | 保証期間・保証範囲の確認 |
| 変更点一覧 | 設計変更・代替品使用の記録 |

!!! danger "図面未更新のまま引き渡されると保全時に事故要因になる"
    「As-built 図面を更新してから引き渡す」をベンダーとの契約・仕様に明記しておく。  
    施工後に図面更新を依頼しても作業費・時間がかかる上、精度が下がる。

---

## ⑥ 運転・保全部門への引継ぎ

工事が終わっても、**現場が安全に使えなければ設備技術部の仕事は終わっていない。**

### 引継ぎ内容

| 項目 | 内容 |
|-----|-----|
| 操作方法 | 正常起動・停止手順 |
| 異常時対応 | 警報発報時の対処手順・連絡先 |
| 点検ポイント | 定期点検項目・頻度・判定基準 |
| 消耗品・予備品 | 品番・調達先・推奨在庫数 |
| 注意事項 | 設備固有の注意点・禁止事項 |

引継ぎが口頭のみの場合、必ず議事録 or チェックシートに残す。  
必要に応じてベンダーによる **説明会・OJT** を実施する。

---

## ⑦ 残課題の是正管理

軽微な手直しや宿題事項を口頭で流すと **「完了扱いだが未完」** になる。

リスト化のフォーマット：

```
No. | 是正内容 | 対応期限 | 担当（ベンダー/自社） | 確認日 | 完了確認者
```

!!! tip "完了の判定を担当者が持つ"
    是正項目の完了確認は設備技術部担当者が行う。ベンダーの自己申告で閉じない。

---

## ⑧ 検収・支払判断

設備技術部の立場では、**以下4点がそろって初めて「検収可」の判断材料が揃う。**

| 条件 | 判定 |
|-----|-----|
| 現場OK（施工品質・外観） | □ |
| 性能OK（試運転・動作確認） | □ |
| 書類OK（図面・記録・マニュアル） | □ |
| 残件OK（是正完了・引継ぎ完了） | □ |

4点すべてに □ がついて初めて経理・購買への検収連絡を行う。

---

## 「工事完了」と「検収」の違い

よくある混同：「工事完了報告 = 検収合格」と勘違いされることがあります。

| 観点 | 工事完了報告 | 検収判断 |
|-----|-----------|--------|
| **主体** | ベンダー（施工者） | 設備技術部（発注者） |
| **タイミング** | 施工完了時 | 試運転～立上げ完了後 |
| **確認内容** | 「仕様どおり施工した」 | 「要求どおり機能・安全」 |
| **以降の責任** | ベンダーは撤収 | 自社（保全部門）へ移管 |
| **支払連携** | まだ支払うべきではない | OK判定後に支払可 |

**ポイント：** ベンダーの「完了宣言」は「あなたの確認が始まる合図」。ここからが設備技術部の仕事。

---

## 設備技術部担当者の視点（リーダーレベル）

チームリーダーとして意識すべき判断軸と残件管理：

### 1. リスク判定軸

- **完了範囲の曖昧さ** → 後戻りコスト最高。最初の1営業日の確認が全て
- **試運転計画の欠落** → 性能合格の判定基準が決まらず、延々ループする
- **図面未更新** → 保全部門が古い図面で対応し、事故要因になる
- **引継ぎ不足** → 運転開始後のトラブル問い合わせが技術部に集中

### 2. 部門間の責任移管ポイント

| タイミング | 移管内容 | チェック |
|----------|--------|---------|
| 現場完成検査後 | 「施工品質OK」を記録（ベンダーとの合意） | ☐ |
| 試運転完了後 | 「性能OK」の試験成績書を収納（保全部へ配付） | ☐ |
| 図面・マニュアル納入時 | As-built 図面の確認・承認（保全部が使える状態か） | ☐ |
| 教育・OJT完了後 | 運転部門・保全部門のサイン取得 | ☐ |

### 3. ベンダーとの契約時に明記すべき項目

- **As-built 図面の納入期限**（施工完了から何営業日以内か）
- **試験成績書の形式**（どの試験項目の成績書が必要か）
- **試運転時の性能基準**（公差、判定基準を明確に）
- **代替品を使用した場合の検証義務**（スペック確認、互換性確認の責任）

---

## 5分類チェックリスト（実務整理）

| 分類 | 担当者がやること |
|-----|--------------|
| **現場** | 完成状態・施工品質・安全確認 |
| **性能** | 試運転・動作・性能確認 |
| **書類** | 図面・試験記録・マニュアル回収 |
| **運用** | 使用部門・保全部門への引継ぎ |
| **管理** | 残件管理・検収判断・支払連携 |
