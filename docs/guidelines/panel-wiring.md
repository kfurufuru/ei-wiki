# 盤内配線作業要領

制御盤・分電盤・計装盤の内部配線作業（新設・増設・改造）の作業要領です。  
安全な配線施工と品質確保のための標準手順を定めます。

---

## 事前確認

!!! warning "作業開始前に必ず確認"
    - [ ] PTW（作業許可証）取得済み
    - [ ] 盤の電源が完全にOFFであることを確認（検電器使用）
    - [ ] 回路図・端子配列図・ケーブルリスト入手済み
    - [ ] 使用するケーブル・端子の仕様確認（耐圧・耐熱・難燃性）
    - [ ] 工具の絶縁被覆確認

### 必要な資格
- 低圧電気取扱特別教育修了（必須）
- 第2種電気工事士以上（電源工事が含まれる場合）

### 必要な工具

| 工具 | 用途 |
|------|------|
| ワイヤストリッパ | 被覆剥き |
| 圧着ペンチ（適合サイズ） | 圧着端子取付 |
| ドライバ（絶縁柄付き） | 端子ネジ締め |
| トルクドライバ（推奨） | 規定トルク管理 |
| デジタルマルチメータ | 導通・絶縁確認 |
| 検電器 | 活電確認 |
| 電線チューブ・ワイヤマーカ | 配線識別 |

---

## 作業前の安全確認

### 検電手順

1. 検電器の動作確認（電源ON、既知の活電部で動作テスト）
2. 盤の主幹ブレーカを確認し、OFFであることを確認
3. 検電器で**主幹ブレーカ二次側（盤内全端子台）**を確認
4. 「停電中・作業中」の警告タグを盤扉・ブレーカに取り付ける

!!! danger "絶対に守ること"
    - 検電確認なしの盤内作業は禁止
    - 他の電源から盤内に電気が供給されている可能性を常に考慮すること
    - UPS・蓄電池回路は主幹OFFでも電圧が残る場合がある

---

## 配線作業手順

### STEP 1：ケーブル引き込み

1. ケーブルグランド（またはブッシング）のサイズを確認し取り付ける
    - 外径に合ったグランドを選定（防水・難燃性の確認）
    - 締め付けは規定トルクで実施（ケーブルへのダメージ防止）
2. ケーブルを引き込み、余長を適切に確保する
    - 端子台への接続分＋将来の接続変更分として100〜150mm余長を確保
3. ケーブルクランプでケーブルを固定する（引っ張りに対する機械的保護）

### STEP 2：電線端末処理

1. ケーブルシース（外装被覆）を剥く
    - 剥き長：端子台高さ＋余長で50〜80mm程度
    - シース剥きでコア絶縁を傷つけないこと
2. シールド付きケーブルの場合、シールド処理を行う
    - シールドをよじってドレン線（アース線）として整形する
    - 接地方法（片端/両端）をループ図・仕様書で確認
3. 芯線の被覆を剥く
    - 剥き長：端子台仕様に合わせる（棒端子使用時は棒端子長さに合わせる）
    - **剥き過ぎ禁止**（露出過大で短絡リスク）

### STEP 3：端子処理

#### 棒端子（フェルール端子）使用の場合（推奨）

1. 剥き長に合った棒端子を選定する（芯線太さ × 剥き長でサイズ確認）
2. 芯線を棒端子に挿入し、専用圧着ペンチで圧着する
3. 圧着後、軽く引っ張って抜けないことを確認する

#### 裸圧着端子使用の場合

1. 電線サイズに合ったYまたはO端子を選定する
2. 被覆ストリップ後、端子に挿入して圧着する
3. 圧着マーク（刻印）が均等に入っていることを確認

!!! warning "電線サイズと端子の適合"
    電線サイズに合わない端子の使用は接触不良・発熱の原因になる。  
    圧着ペンチのダイス選定も電線サイズに合わせること。

### STEP 4：電線識別（マーキング）

1. ワイヤマーカ（電線番号チューブ）を取り付ける
    - 端子台接続前に通しておく
    - 番号は回路図・ケーブルリストの番号と一致させる
2. マーカが読めるように向きを統一する（端子側から番号が読める方向）
3. 同一ループの電線は色分けに従う

| 回路 | 一般的な色分け |
|------|-------------|
| 4-20mA信号（＋） | 赤 |
| 4-20mA信号（−） | 黒 |
| DC24V電源（＋） | 赤 |
| DC24V電源（−/0V） | 青 |
| アース（接地） | 緑/黄緑 |
| AC制御電源 | 赤（L）/ 白（N） |

### STEP 5：端子台への接続

1. 端子台番号が回路図と一致していることを確認する
2. 端子ネジを緩め、棒端子（または裸端子）を挿入する
3. **規定トルクで締め付ける**
    - M3端子：0.5〜0.8 N·m
    - M4端子：0.8〜1.2 N·m
    - M5端子：1.5〜2.0 N·m
    - 推奨：トルクドライバ使用
4. 接続後、電線を軽く引っ張って抜けないことを確認する

!!! warning "過大トルク禁止"
    締めすぎは端子台の割れ・ネジ頭舐めの原因になる。  
    特にフェニックスコンタクト等の細ピッチ端子は締めすぎ注意。

### STEP 6：配線の整形・固定

1. 配線束をケーブルタイ（インシュロック）で適切な間隔でまとめる
    - 間隔：150〜200mm程度
    - 締め付け強さ：電線が変形しない程度
2. 電線を盤内配線ダクトに収める
    - ダクト充填率：80%以下（将来の増設を考慮）
    - 動力線と信号線は**別ダクトに収める**（ノイズ分離）
3. 余長は無理なく曲げてまとめる（曲げ半径：電線外径の5倍以上）

---

## 配線完了後の確認

### 導通確認

1. マルチメータをΩモードに設定
2. 回路図に従い、端子台の同一回路端子間の**導通（低抵抗）**を確認
3. 異なる回路間に**導通がないこと**（短絡なし）を確認

### 絶縁確認

1. 電源を入れる前に、DC500Vメガーで絶縁抵抗を測定する
2. 確認対象：各回路〜アース間
3. **基準値：10MΩ以上**（低い場合は配線を見直す）

## チェックリスト（作業完了時）

- [ ] 全端子の接続完了・トルク確認
- [ ] 電線マーカの取り付け・番号一致確認
- [ ] シールド線の接地処理確認
- [ ] 導通確認完了（全回路）
- [ ] 絶縁抵抗確認完了（10MΩ以上）
- [ ] 警告タグの取り外し
- [ ] 盤扉・カバーの復旧
- [ ] 不要な端子カバーの取り付け
- [ ] 作業記録（配線変更図への反映）

---

## よくある不具合と対処

| 不具合 | 原因 | 対処 |
|--------|------|------|
| 接触不良・断線 | 端子締め付け不足・芯線折れ | 増し締め・端子交換 |
| 短絡 | 被覆剥き過ぎ・端子間隔不足 | 絶縁チューブ追加・再配線 |
| ノイズ混入 | 信号線と動力線の混在 | 配線ダクト分離 |
| 番号違い | 配線時の取り違え | 回路図との照合・是正 |
