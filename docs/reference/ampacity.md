# ケーブル許容電流と補正係数

## 30秒まとめ

実際に流せる電流（実許容電流）は、カタログの基準許容電流をそのまま使えません。周囲温度と多条布設（ケーブルを束ねる本数）で下がります。

```
実許容電流 = 基準許容電流 × 周囲温度補正係数 × 多条布設低減係数
```

- 基準許容電流の数値は [低圧ケーブル](../02-teiatsu/lv-cable.md) の許容電流表を正典（SSOT）として参照してください。本ページは補正係数のSSOTです。
- 本ページの係数を掛けた実許容電流が、負荷電流以上であることを確認します。

!!! warning "基準温度の取り違えに注意"
    許容電流表には「何℃を基準にした値か」（基準周囲温度）が必ず紐づきます。[低圧ケーブル](../02-teiatsu/lv-cable.md) の 600V CV/CVT 表は **40℃ 基準** です。一方、内線規程の絶縁電線（IV 等）許容電流は **30℃ 基準** です。補正係数は「基準温度で 1.0（無補正）」となるため、起点が違えば同じ周囲温度でも係数が変わります。使う表の基準温度を必ず確認してください。

---

## 周囲温度補正係数

周囲温度が基準と異なるときに乗じます。絶縁体の最高許容温度で係数が変わります。計算式は次のとおりです。

```
周囲温度補正係数 k1 = √( (θmax - θa) / (θmax - θ0) )

θmax : 導体の最高許容温度 [℃]（PVC/IV = 60℃、CV・架橋ポリエチレン = 90℃）
θa   : 実際の周囲温度 [℃]
θ0   : 許容電流表の基準周囲温度 [℃]
```

### CV・CVT（架橋ポリエチレン絶縁、最高許容温度 90℃）

[低圧ケーブル](../02-teiatsu/lv-cable.md) の表（40℃ 基準）に乗じる係数です。

| 周囲温度 [℃] | 補正係数 k1 |
|------------|-----------|
| 30 | 1.09 |
| 35 | 1.04 |
| 40（基準） | 1.00 |
| 45 | 0.94 |
| 50 | 0.89 |

### PVC・IV（ビニル絶縁、最高許容温度 60℃）

内線規程の絶縁電線（30℃ 基準）に乗じる係数です。

| 周囲温度 [℃] | 補正係数 k1 |
|------------|-----------|
| 30（基準） | 1.00 |
| 35 | 0.91 |
| 40 | 0.82 |
| 45 | 0.71 |
| 50 | 0.58 |

!!! note "端数処理の考え方"
    上表は上記の計算式による値で、内線規程・JCS の掲載表も同じ √ 式を基礎としています。絶縁電線（IV・最高許容温度 60℃、30℃ 基準）では周囲温度 40℃ の係数が 0.816 と掲載され、本表の 0.82 はその小数第2位への丸めです。本ページの CV 側は安全側に小数第2位を切り捨てています（35℃: 式 1.049 → 表 1.04、45℃: 式 0.949 → 表 0.94）。このため掲載表の丸め方により小数第2位で 0.01 程度前後することがあります。0.01 の差が余裕率に効く厳密設計では、内線規程（JEAC 8001）原典の数表で確認してください。

## 多条布設低減係数（電流減少係数）

同一電線管・ダクト内、または同一ラック上に複数条を密着布設すると放熱が悪化し、許容電流が下がります。内線規程の絶縁電線（電線管内）における電流減少係数を示します。

| 同一管・ダクト内の条数 | 電流減少係数 k2 |
|---------------------|--------------|
| 3 以下 | 0.70 |
| 4 | 0.63 |
| 5〜6 | 0.56 |
| 7〜15 | 0.49 |
| 16〜40 | 0.43 |
| 41〜60 | 0.39 |

!!! warning "布設形態で係数が変わる"
    - 上表は内線規程の**絶縁電線を電線管内**に収める場合の値です。基準となる許容電流が「気中単独」の値であることが前提です。
    - **気中（碍子布設・単独）** は低減なし（k2 = 1.0）。
    - **ケーブルラック（トレイ）の多条・多段積み** は、上表（電線管内）とは別体系の低減係数（JCS 0168-2／0168-3 系のケーブル多条布設低減率）を適用します。条数だけでなく段数・水平離隔（ケーブル中心間距離）で値が変わります。
    - [低圧ケーブル](../02-teiatsu/lv-cable.md) の管路・気中・ラック別の基準値は「単独条」の値です。複数条まとめる場合は本係数を追加で乗じます。

### ケーブルラック・気中多条布設の低減率（目安）

気中・ラック布設の低減率 η は、ケーブル中心間距離 S とケーブル仕上外径 d の比、および段数で決まります。以下は JCS 0168-2／0168-3 系およびメーカー技術資料に基づく代表的な目安です。厳密値は条数・外径・段数ごとに変わるため、原典またはメーカーの多条布設低減率表で個別に確認してください。

| 布設形態 | 離隔（中心間距離 S） | 電流低減率 η（目安） |
|---------|------------------|------------------|
| 単層・密着 | S = d（隣接ケーブルが接触） | 約 0.70 |
| 単層・離隔 | S = 2d（外径分あけ） | 約 0.85 |
| 単層・離隔 | S ≥ 3d（外径3倍以上） | 約 0.90〜0.95 |
| 2段積み・密着 | S = d | 約 0.50〜0.65 |
| 3段以上・密着 | S = d | さらに低下（要個別計算） |

- 離隔を広げるほど η は 1.0 に近づきます。逆に段を重ねるほど下段・中段の放熱が悪化して η は下がります。
- 上表の η は「単独条・気中」の基準許容電流に乗じる値です。[低圧ケーブル](../02-teiatsu/lv-cable.md) のラック敷設値（単独条）にさらに多条・多段分として乗じます。

## 適用手順

1. [低圧ケーブル](../02-teiatsu/lv-cable.md) の許容電流表から、ケーブル種類・断面積・布設方法に対応する**基準許容電流**を読む（表の基準周囲温度を確認）。
2. 実際の周囲温度に対する**周囲温度補正係数 k1** を上表から求める（絶縁種別に注意）。
3. 同一管・ラックの条数に対する**多条布設低減係数 k2** を求める（単独条なら 1.0）。
4. `実許容電流 = 基準許容電流 × k1 × k2` を計算し、**負荷電流と比較**する。余裕率 =（実許容電流 / 負荷電流 − 1）× 100 [%] が確保されているか確認する。

!!! tip "余裕率の目安"
    連続負荷では実許容電流に対し 20% 程度の余裕を見込むと、将来増設や測定誤差を吸収しやすくなります。

## 計算例

### 例1: CV 38sq、高温環境（温度補正のみ）

- 基準許容電流: CV 38sq 管路敷設 = **105 A**（[低圧ケーブル](../02-teiatsu/lv-cable.md) の表、40℃ 基準）
- 周囲温度 50℃ → CV の k1 = **0.89**
- 単独条布設 → k2 = 1.0

```
実許容電流 = 105 × 0.89 × 1.0 = 93.4 A ≒ 93 A

負荷電流 80 A に対し 余裕率 =（93 / 80 − 1）× 100 = 16%  → OK
```

40℃基準の 105 A をそのまま使うと過負荷を見落とすため、高温盤内・屋外直射環境では温度補正が必須です。

### 例2: CV 38sq、複数条を同一管路（温度補正＋多条低減）

- 基準許容電流: CV 38sq 管路敷設 = **105 A**（40℃ 基準）
- 周囲温度 35℃ → CV の k1 = **1.04**
- 同一管に 4 条布設 → k2 = **0.63**

```
実許容電流 = 105 × 1.04 × 0.63 = 68.8 A ≒ 69 A

負荷電流 60 A に対し 余裕率 =（69 / 60 − 1）× 100 = 15%  → OK
```

!!! note "条数の数え方"
    内線規程の許容電流表の注記により、**中性線は条数に数えません**。単相 2 線式は 2 条、単相 3 線式は「単心 2 個より」（中性線を除き 2 条相当）、三相 3 線式は 3 条（単心 3 個より）として扱います。三相 4 線式は中性線を除いて「単心 3 個より」の値を用います。多心ケーブル 1 本は、電流を流す線心数で数えます（例: CV 3 芯 = 3 条相当、CVT = 3 条相当）。この数え方に基づく条数を、電線管内なら本ページの電流減少係数表、気中・ラックなら上記の多条布設低減率に当てはめます。

## 現場の落とし穴

- **トレイの多段積み**: ラックを上下に重ねると下段・中段ほど放熱が悪化。単なる条数だけでなく段数の低減も効く（上記「ケーブルラック・気中多条布設の低減率」参照）。厳密値は原典の多段係数を確認する。
- **日射（太陽直射）**: 屋外の気中布設は日射で導体温度が上がる。気中値より低い直射補正が必要。
- **盤内温度上昇**: 盤内は外気より 10〜20℃ 高くなることがある。周囲温度は「盤内実温度」で評価する。
- **将来増設分の条数見込み**: 竣工時は少条でも、後日ケーブルを追加すると条数が増えて既設分の許容電流も下がる。将来本数を見込んで係数を選ぶ。
- **基準温度の取り違え**: 40℃ 基準の CV 表に 30℃ 基準の係数を掛ける等の混同は過大・過小の両方を招く。表の基準温度と絶縁種別を必ず一致させる。
