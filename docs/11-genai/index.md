## なぜ今、生成AIを使うのか

2023〜2024年にかけて、ChatGPT・Copilot・Claudeといった生成AIが急速に実用レベルに達した。単なる「便利ツール」ではなく、**業務の質とスピードを根本から変えるインフラ**になりつつある。

電気計装の現場で特に効果が高い場面：

- **文書作成の高速化**：仕様書・稟議書・報告書のドラフトを数分で生成
- **知識の壁打ち**：規格や技術情報の一次調査を会話形式で素早く完了
- **トラブル解析の補助**：症状を言語化して原因候補をAIに列挙させる
- **翻訳・読解**：英語のメーカー資料・IEC規格をその場で日本語化

!!! note "「AIに仕事を奪われる」より「AIを使いこなせる人と使えない人の差が広がる」"
    活用できる人が少ない今が、差をつけるチャンス。このWikiを足がかりに、チーム全員でレベルを上げていく。

## セキュリティの基本（先に確認）

活用を始める前に、[注意事項・情報セキュリティ](security.md) を必ず一読すること。

**一言でまとめると：**
- M365 Copilot（会社契約）→ 社内情報を入力してよい
- ChatGPT / Claude.ai（個人アカウント）→ 社外秘・個人情報は入力禁止
