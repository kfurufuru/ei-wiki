# ケーブルトレイ施工要領

ケーブルトレイ（ラックトレイ・メッシュトレイ）の設置・ケーブル延線の施工要領です。  
新設工事・増設工事時の標準手順を定めます。

---

## 事前確認

!!! warning "作業開始前に必ず確認"
    - [ ] PTW（作業許可証）取得済み
    - [ ] 施工図（ケーブルルート図）入手済み
    - [ ] ケーブルラック設計図（幅・高さ・支持間隔）確認済み
    - [ ] 既設トレイの充填率確認（追加可能かの確認）
    - [ ] 高所作業の場合：高所作業特別教育修了、ハーネス準備
    - [ ] 工場所轄部門との工事調整完了

### 必要な資格
- 高所作業（2m以上）：特別教育修了
- 玉掛け・移動式クレーン（重量物搬入時）：各資格

### 必要な機器・工具

| 工具・資材 | 用途 |
|-----------|------|
| 電動ドリル・インパクトドライバ | ブラケット固定 |
| アンカーボルト（適合サイズ） | 天井・壁への固定 |
| 水準器 | 水平確認 |
| メジャー・チョークライン | 位置出し |
| ケーブルタイ（インシュロック） | ケーブル固定 |
| 延線用ローラ | ケーブル延線時の保護 |
| 安全帯・ハーネス | 高所作業時 |

## ケーブルトレイの種類と選定

| 種類 | 特徴 | 適用場所 |
|------|------|---------|
| ラダートレイ | 強度が高い・大径ケーブル向き | 動力ケーブルメインルート |
| メッシュトレイ | 軽量・通風性・信号線向き | 計装ケーブル・制御ケーブル |
| ソリッドボトムトレイ | 防塵・油散布環境向き | プロセス設備周辺 |
| コンジット（配管） | 単独ルート・防爆エリア | 防爆区域・露出部分 |

!!! info "動力線と信号線の分離"
    動力ケーブルと計装・制御信号線は**別のトレイに分離**して布設する。  
    同一トレイに収める場合は金属セパレータで分離し、間隔を150mm以上確保する。

---

## 施工手順

### STEP 1：ルート確認・位置出し

1. 施工図（ケーブルルート図）と現場を照合し、以下を確認する
    - 既設配管・ダクト・構造物との干渉
    - 他工事との取り合い（電気・計装・配管・空調）
    - 保守点検の通路確保（通路幅600mm以上を確保）
2. チョークラインで壁・天井に支持点の位置を墨出しする
    - 支持間隔：**1.5m以下**（JIS基準）、重量物の場合は1.0m以下
3. 水準器で水平・垂直を確認しながら位置出し

### STEP 2：支持物（ブラケット・ハンガー）の取り付け

1. 取付位置にアンカー穴を開ける
    - コンクリート：振動ドリル使用、穴径はアンカー仕様に合わせる
    - 鉄骨：ドリルビットで穴開け後ボルト固定
2. アンカーボルトを打ち込み、規定トルクで締め付ける
3. ブラケット・ハンガーを取り付け、水平を確認する
4. 支持物の荷重強度が設計荷重（ケーブル重量＋トレイ重量）に対して十分であることを確認

!!! warning "アンカー強度の確認"
    アンカー選定は施工図の仕様に従う。独断でサイズを変えないこと。  
    不安な場合は電計チームリーダーに確認。

### STEP 3：トレイ本体の設置

1. トレイを支持物に乗せ、**ボルト・ナットで固定**する
    - 緩み止めワッシャ（スプリングワッシャ）を使用する
    - 規定トルクで締め付ける（M8ボルト：20〜25 N·m目安）
2. トレイ同士の接続部（継手）を取り付け、**ボルト固定**する
    - 継手はトレイと同材質・同メーカーを使用（強度確保）
3. 曲がり・T分岐部は専用の曲がり継手・T型継手を使用する
    - 無理な曲げ加工は強度低下・ケーブル損傷の原因
4. 全体の水平・垂直を再確認する

### STEP 4：接地ボンディング

1. トレイ接続部に**ボンディングジャンパ**を取り付ける
    - 継手のボルト締結だけでは電気的連続性が不十分な場合がある
    - 金属製トレイは接地（D種以上）が必要
2. トレイ系統の末端に接地線を接続する
3. 接地抵抗を測定し、基準値（D種：100Ω以下）を満たすことを確認する

!!! info "アルミトレイの接地"
    アルミトレイと鉄部品を直接接触させると電食（電気化学腐食）が発生する。  
    異種金属間には絶縁材（スリーブワッシャ等）を挟むこと。

### STEP 5：ケーブル延線

1. 延線前にトレイ上・内側に**バリ・鋭利な突起がないことを確認**する
    - あればヤスリで除去し、保護テープを巻く
2. 大径ケーブルは延線用ローラを使用して引きずらないように延線する
3. ケーブルを延線しながら、**トレイ上で束ねすぎない**ように整列する
    - 多段積みにしない（発熱・許容電流低下）
    - トレイ幅の80%以下が目安

#### ケーブル配置の原則

| 上段 | 中段 | 下段 |
|------|------|------|
| 計装・信号線 | 制御ケーブル | 動力ケーブル |

（セパレータで分離する場合は同段でも可）

4. ケーブルに**ケーブルタグ**を取り付ける
    - 両端（始点・終点）と中間（30m毎）に取り付ける
    - タグ番号はケーブルリストと一致させる

### STEP 6：ケーブル固定

1. ケーブルをケーブルタイで固定する
    - 固定間隔：水平部は**1.5m以下**、垂直部は**1.0m以下**
    - 締め付けでケーブルが変形しない程度（目安：余裕1mm程度）
2. 垂直部のケーブルは**ケーブルクリート**等で確実に支持する
    - ケーブルの自重でコネクタ・端子台に力がかかると接触不良の原因
3. 機器接続部付近は余長を適切に確保する
    - 引っ張りによる断線防止
    - 機器の微小な移動・振動への対応

### STEP 7：防火措置（延焼防止）

1. 防火区画（壁・床貫通部）を通過する場合は**防火措置**を施す
    - 防火パテ・防火スリーブ等を使用
    - 建築基準法・消防法に基づく施工
2. 貫通後は**防火区画図**に反映する

## 施工完了後の確認

- [ ] 支持物の取り付け確認（ボルト緩み・脱落なし）
- [ ] 接地ボンディング確認
- [ ] ケーブル固定状態確認（タイ切れ・緩みなし）
- [ ] ケーブルタグ取り付け確認
- [ ] 防火措置確認（貫通部）
- [ ] トレイ充填率確認（80%以下）
- [ ] 通路・保守スペース確保確認

---

## トレイ充填率の計算

充填率が80%を超えると放熱不足による許容電流低下が生じる。

```
充填率 (%) = (収容ケーブルの外径断面積の合計) ÷ (トレイの内幅 × 内高さ) × 100
```

例：トレイ内幅200mm、内高さ60mm、ケーブル断面積合計8,000mm²  
→ 充填率 = 8,000 ÷ (200 × 60) × 100 = 66.7%（OK）

## よくある施工不良と対処

| 不具合 | 原因 | 対処 |
|--------|------|------|
| トレイの歪み・傾き | 支持間隔過大・固定不足 | 中間支持追加・ボルト増し締め |
| ケーブル損傷 | バリによる傷・無理な曲げ | バリ処理・ローラ使用 |
| 接地不良 | ボンディング未施工 | ジャンパ線追加 |
| 電食（腐食） | 異種金属接触 | 絶縁ワッシャ使用 |
| 過剰充填 | 設計変更未反映 | トレイ幅拡大または別ルート追加 |
