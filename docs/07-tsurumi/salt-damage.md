# 塩害・腐食環境への対応

## 鶴見大黒地区の腐食環境

大黒町は埋立地の「島」であり、東京湾に四方を囲まれている。塩害の特徴：

- 海風の方向が一定でなく、全方向から塩分粒子が付着
- 乾燥→結露を繰り返すことで塩分が濃縮・蓄積
- 汎用工場の「海岸から500m以内」基準では不十分で、より厳しい対応が必要

## 材質選定基準（鶴見標準）

| 部位 | 汎用工場 | 鶴見大黒地区 |
|------|---------|------------|
| 計装チューブ | SUS304 | **SUS316L 必須** |
| ジャンクションボックス | 鉄製（塗装） | **SUS316L または FRP** |
| ケーブルラック | 溶融亜鉛メッキ | **SUS316L またはGRPラック** |
| ボルト・ナット | 鉄製メッキ | **SUS316 またはA4-70** |
| 盤外装 | 鉄製（塗装） | **SUS316L または FRP** |
| 配管サポート | 一般鉄鋼 | **SUS316L** |

!!! warning "SUS304 では不十分"
    塩化物環境ではSUS304の隙間腐食・孔食が数年で発生する。
    鶴見大黒地区では**SUS316L**（Mo含有）を標準として選定すること。

## IP等級の基準（鶴見標準）

| 設置場所 | 最低IP等級 | 推奨 |
|---------|-----------|------|
| 屋外（地上1m以上） | IP65 | IP66 |
| 屋外（地上1m以下、浸水リスクあり） | IP66 | IP67 |
| 屋内（換気あり） | IP54 | IP65 |
| 防爆エリア屋外 | IP65 + Ex認証 | IP66 + Ex |

## 保全上の注意点

- **年1回の塩分付着確認**：端子台・リレー接点の接触抵抗測定
- **塩害起因の絶縁低下**：降雨後に絶縁抵抗が回復するが、晴天時の繰り返し測定でトレンドを把握
- **CUI（断熱材下腐食）**：保温配管の外面腐食は目視では発見困難。3〜5年周期でサンプリング調査を実施
- **接続部の腐食**：圧着端子・コネクタは銀メッキ品（金メッキ品が理想）を使用

## よくあるトラブル（鶴見固有）

!!! failure "やりがちなNG"
    - **NG**: 汎用品（鉄製塗装BOX）を「安いから」と選定 → 3年で塗装剥離・腐食穿孔
    - **NG**: SUS304ボルトをSUS316L筐体に使用 → 異種金属腐食（ガルバニック腐食）
    - **NG**: 塩害対策を「塗料で補う」で済ませる → 傷・チッピングから急速腐食

!!! tip "塩害点検のタイミング"
    台風・強風後の翌日に塩分付着量が最大になる。
    台風シーズン後（9〜10月）の定期点検に塩害確認を組み込む。
