# 操作盤・表示灯

## 30秒まとめ

操作ボタンと表示灯の色は JIS / IEC 規格で決まっている。非常停止は赤（キノコ型）、起動は白または緑が基本。HMI 画面はアラームを画面上部・左側に配置し、重要情報が埋もれない設計にする。防爆エリアの操作機器は Ex 認証品を使用する。

---

## 操作ボタン配色規格

| 操作 | 色 | 根拠（JIS C 0448 / IEC 60073） |
|------|----|-----------------------------|
| 起動・始動 | 白（または緑） | 緑は「安全」「進行」の意味 |
| 停止 | 赤（または黒） | 赤は「危険」「停止」 |
| 非常停止 | 赤（キノコ型ヘッド）・黄色背景プレート | IEC 60947-5-5 準拠 |
| 点検・確認 | 青（または白） | — |
| リセット | 青 | — |

!!! warning "白・緑の使い分け"
    JIS C 0448:2011 では起動ボタンを「白」に統一し、緑は状態表示（OK / 運転中）に使用する方向。ただし既設設備との統一性を優先する場合は現場の基準に合わせる。

---

## 表示灯の色規格（JIS C 0448 / IEC 60073）

| 色 | 意味 | 典型的な用途 |
|----|------|------------|
| 赤 | 危険・警告・停止 | 異常・故障・非常停止動作中 |
| 黄（アンバー） | 注意・警戒 | 警告（軽度の異常）・予備加熱中 |
| 緑 | 正常・安全 | 運転中・電源 ON・弁開 |
| 青 | 強制・特定状態 | 遠隔モード中・手動選択中 |
| 白 | 電源・一般状態 | 電源投入中・その他 |

```
典型的な表示灯配置（ポンプ操作盤）:
[電源 白] [運転中 緑] [停止中 赤] [異常 赤点滅] [遠隔 青]
```

## タッチパネル HMI 画面設計のポイント

### 優先度別配置原則

| 優先度 | 情報種別 | 推奨配置 |
|--------|---------|---------|
| 最高 | アラーム・異常 | 画面上部または常時表示エリア |
| 高 | 主要プロセス値（温度・圧力・流量） | 画面中央・大きな数値で表示 |
| 中 | 弁・機器の開閉状態 | プロセスフローに沿って配置 |
| 低 | 設定値・パラメータ | サブ画面に移動 |

### 設計チェックポイント

| 項目 | 基準 |
|------|------|
| 数値の視認性 | 最小文字サイズ 14pt 以上（現場の明るさを考慮） |
| 色だけで状態を区別しない | 色覚異常者への配慮：形・記号も使う |
| タッチターゲットサイズ | 最小 10mm × 10mm（作業用手袋着用を考慮） |
| アラームの優先付け | 設備重要度 A/B/C に連動したアラームレベル設定 |
| 操作確認ダイアログ | 重要操作（起動・停止）は確認ポップアップを挿入 |

## IP 等級の選定

| 設置環境 | 推奨 IP 等級 | 根拠 |
|---------|------------|------|
| 屋内・乾燥 | IP42 以上 | 異物・水滴の侵入防止 |
| 屋外（雨ざらし） | IP65 以上 | 粉塵完全遮断・水噴流 |
| 腐食雰囲気（化学プラント） | IP55〜IP65 | SUS 製エンクロージャ推奨 |
| 洗浄あり（食品・薬品） | IP66〜IP69K | 高圧洗浄水への耐水 |
| 水没可能性 | IP67〜IP68 | 水中浸漬 |

IP 表記の読み方：`IP X Y`
- X（第 1 数字）: 固体粉塵に対する保護（0〜6）
- Y（第 2 数字）: 水に対する保護（0〜9K）

## 化学プラント固有：防爆エリアの操作機器

!!! danger "防爆エリアでは Ex 認証品を使用"
    危険場所（Zone 0/1/2、グループ II）に設置する操作機器・表示灯は防爆認証（Ex 認証）品を使用すること。認証なし機器の設置は法令違反。

| 防爆構造 | 記号 | 特徴 | 適用 Zone |
|---------|------|------|----------|
| 耐圧防爆 | Ex d | 爆発を盤内に封じ込める | Zone 1, 2 |
| 安全増防爆 | Ex e | スパーク・高温部なし | Zone 1, 2 |
| 本質安全防爆 | Ex ia/ib | エネルギー制限 | Zone 0, 1, 2 |
| 粉塵防爆 | Ex t | 粉塵爆発環境 | Zone 21, 22 |

**機器選定時の確認事項：**

1. 危険場所の区分（Zone 0/1/2）と温度クラス（T1〜T6）を確認
2. 機器の Ex 認証証書（ATEX / JIS C 60079）を入手
3. 設置後は「危険場所設置記録」に登録
4. メンテナンス時は Ex 認証品と同等品でのみ交換
