# 安全管理・作業許可

> 電気作業における安全管理は「知っている」だけでなく、**手順を守り続けること**が全て。
> このセクションでは、鶴見工場の実運用に即した安全管理の基本を整理する。

## 安全の基本原則

!!! danger "絶対厳守"
    - 作業許可書なしの電気作業は禁止
    - LOTO未実施での機器内作業は禁止
    - 検電未実施での接触作業は禁止

!!! info "このセクションの活用方法"
    - 新入メンバーの安全教育資料として使用可能
    - KY活動・ツールボックスミーティングの参考資料として活用
    - 変更管理（MOC）時のリスクアセスメントのチェックリストとして参照
