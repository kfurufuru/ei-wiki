# 保安規程の管理

## 保安規程とは

自家用電気工作物の設置者が、電気工作物の工事・維持・運用に関する保安を確保するために作成する社内規程。

**法的根拠**: 電気事業法第42条

```
電気事業法第42条（保安規程）
設置者は、工事・維持・運用に関する保安を確保するため、
主務省令で定めるところにより保安規程を定め、
使用開始前に届け出なければならない。
```

---

## 保安規程の必須記載事項（電気事業法施行規則第50条）

| 項番 | 必須記載事項 |
|------|------------|
| 1 | 電気工作物の工事・維持・運用に関する業務を管理する者の職務・組織 |
| 2 | 電気工作物の工事・維持・運用に従事する者に対する保安教育 |
| 3 | 電気工作物の工事・維持・運用に関する保安のための巡視・点検・検査 |
| 4 | 電気工作物の運転・操作に関する保安 |
| 5 | 発電所・変電所等の運転・操作に関する保安 |
| 6 | 電気工作物に異常がある場合の連絡・措置 |
| 7 | 電気工事の保安 |
| 8 | 電気火災・感電等の事故が発生した場合の措置 |
| 9 | 電気工作物の工事・維持・運用に関する記録の保管 |
| 10 | 保安規程の遵守を確保するための措置 |

### 実務的な重要ポイント

**職務・組織（項番1）**: 電気主任技術者の職務範囲と指揮命令系統を明確化。
「誰が主任技術者か」「主任技術者が不在の時は誰が代行するか」を明記する。

**点検スケジュール（項番3）**: 月次・年次の点検頻度・方法・使用計器を具体的に記載。
曖昧な表現（「適宜」「随時」）は避け、頻度を数字で明記する。

**事故連絡（項番6）**: 事故発生時の連絡先（電力会社・産業保安監督部・社内報告先）と連絡手順を記載。

## 保安規程の変更届出区分

### 届出が必要な変更（軽微変更に該当しない場合）

| 変更内容 | 届出要否 |
|---------|---------|
| 電気主任技術者の変更（選任・解任） | **要届出**（別途選解任届出） |
| 保安監督の組織体制の変更 | **要届出** |
| 点検頻度の変更（緩和・強化） | **要届出** |
| 事故時の連絡先・連絡体制の変更 | **要届出** |
| 対象設備の増減（大規模変更） | **要届出** |

### 届出不要な変更（軽微変更）

| 変更内容 | 届出要否 |
|---------|---------|
| 担当者名の変更（肩書・組織は不変） | **不要**（社内保管版の更新のみ） |
| 誤字・脱字の修正 | **不要** |
| 規程番号・様式の整備 | **不要** |

!!! warning "軽微変更の範囲は慎重に判断する"
    「軽微変更」の判断が難しい場合は、所轄の産業保安監督部に事前確認することを推奨。
    届出漏れが後で判明すると是正指導の対象になる。

### 変更届出の手順

```
1. 変更内容の決定
    ↓
2. 変更届出書の作成（様式第50号）
    ↓
3. 添付書類の準備
    （変更後の保安規程の全文または変更箇所）
    ↓
4. 産業保安監督部へ提出
    （経産省電子申請システム e-Gov 経由または郵送）
    ↓
5. 受理確認・控えの保管
    ↓
6. 社内版保安規程の更新・周知
```

## 保安規程に基づく点検スケジュール管理

### 年間点検カレンダーの整備

保安規程に記載した点検頻度と実施日を管理する年間カレンダーを作成・更新する。

| 点検種別 | 頻度 | 担当 | 記録様式 |
|---------|------|------|---------|
| 日常巡視 | 毎日（運転当番） | オペレーター | 運転日誌 |
| 月次点検 | 毎月（第○週） | 電計チーム | 月次点検記録表 |
| 年次点検（停電） | 年1回（○月） | 電計チーム＋外部業者 | 年次点検報告書 |
| 精密点検（変圧器油分析等） | 3〜5年毎 | 外部業者 | 精密点検報告書 |

### 記録の保管期間

| 記録種別 | 保管期間 |
|---------|---------|
| 月次点検記録 | 3年以上（保安規程で定める） |
| 年次点検報告書 | 3年以上 |
| 事故記録 | 5年以上（事故報告書の控えを含む） |
| 絶縁抵抗・接地抵抗測定記録 | 3年以上 |

## 実態と保安規程の乖離リスク（定期見直しの重要性）

### よくある乖離パターン

| 乖離の種類 | 例 |
|-----------|---|
| 組織変更の反映漏れ | 保安規程上の担当部署名が古い組織名のまま |
| 点検頻度の形骸化 | 規程では月次と書いてあるが実際は2〜3か月に1回 |
| 設備追加の未反映 | 新設した変圧器・発電機が保安規程の対象設備リストに未記載 |
| 連絡先の陳腐化 | 事故時連絡先が退職した担当者のまま |
| 外部委託範囲の不一致 | 外部保安法人に委託したが保安規程が直営前提のまま |

### 保安監督部の立入検査で指摘されやすい点

1. 保安規程と実態の乖離（特に点検記録）
2. 電気主任技術者が実際に監督を行っていない（名義だけ）
3. 記録の保管期間不足
4. 工事計画届出が必要な工事を届出なしに実施

### 定期見直しサイクル（推奨）

```
毎年: 担当者名・連絡先・設備リストの更新確認
3年毎: 点検実施状況との整合確認、設備変更の全体反映
10年毎 or 大規模設備変更時: 規程全文の抜本的な見直し
```

!!! tip "設備投資案件との連動"
    大型設備を新設・廃棄する際は、保安規程の対象設備リスト更新と届出要否の確認をセットで行う。
    設備投資プロジェクトのチェックリストに「保安規程の更新」項目を入れておくと漏れが防げる。
